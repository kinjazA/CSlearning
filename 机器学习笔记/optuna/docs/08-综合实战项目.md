# 第8章：综合实战项目

## 本章目标
将前7章的知识整合，完成两个端到端的超参数优化项目：一个基于 sklearn，一个基于 PyTorch。

---

## 8.1 项目要求

完成以下两个项目，每个项目需要：

1. **至少 5 个超参数**的搜索空间
2. 使用 **TPE Sampler** + **Hyperband Pruner**
3. 运行 **至少 100 个 trials**
4. 保存最佳参数和最佳模型
5. 用 **Dashboard** 分析结果
6. 撰写简短的实验报告

---

## 8.2 项目一：sklearn 房价预测（回归）

### 数据集
使用 `sklearn.datasets.fetch_california_housing` 或 Kaggle 的 House Prices 数据集。

### 搜索空间建议

| 参数 | 类型 | 范围 |
|------|------|------|
| model_type | categorical | ['RandomForest', 'GradientBoosting', 'XGBoost'] |
| n_estimators | int | 50 ~ 500 |
| max_depth | int | 3 ~ 20 |
| learning_rate | float | 1e-3 ~ 0.3 (log) |
| subsample | float | 0.5 ~ 1.0 |
| scaler | categorical | ['Standard', 'MinMax', 'None'] |

### 评估指标
- 主要指标：RMSE（均方根误差）
- 可选：同时优化训练时间（多目标）

### 代码结构
见 `code/08_project_sklearn.py`

---

## 8.3 项目二：PyTorch CIFAR-10 图像分类

### 数据集
使用 `torchvision.datasets.CIFAR10`。

### 搜索空间建议

| 参数 | 类型 | 范围 |
|------|------|------|
| model_type | categorical | ['ResNet18', 'CustomCNN'] |
| lr | float | 1e-4 ~ 1e-1 (log) |
| batch_size | categorical | [64, 128, 256] |
| optimizer | categorical | ['Adam', 'SGD'] |
| weight_decay | float | 1e-5 ~ 1e-2 (log) |
| scheduler | categorical | ['None', 'Cosine', 'Step'] |
| dropout | float | 0.0 ~ 0.5 |

### 评估指标
- 主要指标：验证集准确率
- 可选：同时优化模型参数量（多目标）

### 代码结构
见 `code/08_project_pytorch.py`

## 8.5 进阶挑战

完成基础项目后，尝试以下挑战：

1. **多目标优化**：同时优化准确率和推理速度，绘制 Pareto 前沿
2. **条件搜索空间**：ResNet18 不需要 dropout，CustomCNN 需要
3. **分布式搜索**：在多台机器上并行运行（如果用云服务器）
4. **早停策略对比**：对比 Median、Hyperband、无剪枝的效果
5. **集成学习**：用 Optuna 找到的最佳参数训练多个模型，做集成
