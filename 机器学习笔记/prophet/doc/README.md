# Prophet 时间序列预测 — 系统学习大纲

> **目标**：从零掌握 Facebook Prophet 库，达到能在实际开发场景中独立完成时间序列预测任务的水平。

---

## 课程设计理念

本课程按 **认知 → 建模 → 诊断 → 工程化** 四条主线递进，共 13 章。每章包含三个层次：

| 层次 | 说明 |
|------|------|
| **概念层** | 为什么需要、解决了什么问题、数学直觉 |
| **API 层** | 参数含义、调用方式、输入输出契约 |
| **实战层** | 代码示例、调参经验、常见踩坑与规避 |

## 适用人群

- 有 Python 基础，想系统学习时间序列预测的开发者
- 需要在业务中落地预测需求的数据分析师 / 算法工程师
- 已用过 Prophet 但想深入理解其内部机制和最佳实践的学习者

## 环境要求

```bash
python >= 3.10
prophet >= 1.1
pandas >= 1.3
matplotlib >= 3.5
scikit-learn >= 1.0
```

本课程使用 [uv](https://docs.astral.sh/uv/) 管理虚拟环境和依赖：

```bash
# 1. 安装 uv（如未安装）
# macOS / Linux
curl -LsSf https://astral.sh/uv/install.sh | sh
# Windows PowerShell
powershell -ExecutionPolicy ByPass -c "irm https://astral.sh/uv/install.ps1 | iex"

# 2. 创建项目虚拟环境（在 prophet 学习目录下）
uv venv --python 3.10

# 3. 激活环境
# macOS / Linux: source .venv/bin/activate
# Windows:       .venv\Scripts\activate

# 4. 安装依赖
uv pip install prophet pandas matplotlib scikit-learn jupyter
```

## 学习建议

1. **先看笔记，再跑代码**：每章先通读 MD 理解概念，然后执行对应代码验证理解
2. **改参数观察变化**：跑通代码后，尝试修改关键参数观察预测结果的变化
3. **用自己的数据验证**：学完第 8 章后，找一份自己的业务数据跑一遍完整流程
4. **对照官方文档**：本笔记侧重实战经验，细节定义参考 [Prophet 官方文档](https://facebook.github.io/prophet/)
