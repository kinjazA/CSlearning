# 01 索引切片

> * ndarray 数组可以基于 0 - n 的下标进行索引，切片对象可以通过内置的 slice 函数，并设置 start, stop 及 step 参数进行，从原数组中切割出一个新数组

## 01.1 基本索引

> * 通过整数索引直接访问元素，索引是从0开始的

```python
import numpy as np
arr = np.random.randint(1,100,20)
print(arr[4]) # 这样就是获取第5个元素
print(arr[2:5]) # 这里是左闭右开，也就是获取了第3个元素到第4个元素
print(arr[slice(2,5)])  # 和上面等价
print(arr[:]) # 这样就是获取全部
print(arr[arr>10]) # 这个就是利用布尔条件来获取全部大于10的元素
print(arr[(arr>10) & (arr<70) ])
```

```python
arr = np.random.randint(1,100,(5,7))  #5行7列的矩阵
print(arr[2,3])  # 获取第3行，第4列的元素值
print(arr[1,:])  # 这样表示要第2行的，列全要
print(arr[1,2:5])  # 表示第2行的，第2,3,4列的元素，也就是获取了3个元素
print(arr[arr>20]) # 这样返回的会是一个元素大于20的一维数组
print(arr[2,:][arr[2,:] > 20]) # 这样就是针对第3行来找大于20的元素，嵌套
```

> * 切片还可以包括省略号 **…**

```python
import numpy as np
 
a = np.array([[1,2,3],[3,4,5],[4,5,6]])  
print (a[...,1])   # 第2列元素
print (a[1,...])   # 第2行元素
print (a[...,1:])  # 第2列及剩下的所有元素
```

# 02 ndarray的运算

## 02.1 广播机制

> * 广播(Broadcast)是 numpy 对不同形状(shape)的数组进行数值计算的方式， 对数组的算术运算通常在相应的元素上进行
> * 当运算中的 2 个数组的形状不同时，numpy 将自动触发广播机制

```python
import numpy as np 
 
a = np.array([[ 0, 0, 0],
           [10,10,10],
           [20,20,20],
           [30,30,30]])
b = np.array([0,1,2])
print(a + b)
#[[ 0  1  2]
# [10 11 12]
# [20 21 22]
# [30 31 32]]

```

> * 对两个数组，分别比较他们的每一个维度（若其中一个数组没有当前维度则忽略），满足：
>
>    - 数组拥有相同形状
>    - 当前维度的值相等
>    - 当前维度的值有一个是 1
>
>    若条件不满足，抛出 **"ValueError: frames are not aligned"** 异常

> * 也就是说，广播只在 **某一维是 1 或者两个维度一样大** 时才行。一个三行四列和四行三列的数组运算时，没有广播机制，会抛出报错
