## 🚀 学习路径建议

1. **顺序学习** — 章节按依赖关系排列，建议从头到尾
2. **重点章节** — 第 02 章（序列化）、第 05 章（缓存）、第 07 章（并行）是最核心的三章，工作里最常用
3. **每章节奏** — 先读 doc 理解原理 → 再跑 code 验证 → 自己改参数看效果
4. **第 10 章** — 融合前三模块的综合案例，建议前三章理解后再做

## 🐍 环境管理（uv）

本项目使用 [uv](https://docs.astral.sh/uv/) 管理虚拟环境和依赖。

```bash
# 安装 uv（如尚未安装）
powershell -c "irm https://astral.sh/uv/install.ps1 | iex"

# 初始化环境（首次或 clone 后）
cd joblib/
uv sync           # 核心依赖
uv sync --extra full  # 含 lz4 / zstandard / tqdm / ipython

# 运行代码
uv run python code/01-概述与安装.py

# 启动交互式环境
uv run ipython
```

**已安装依赖（37 个包）**：

| 用途 | 包 | 版本 |
|------|-----|------|
| 核心学习对象 | joblib | 1.5.3 |
| 数值计算 | numpy | 2.5.0 |
| 机器学习 | scikit-learn | 1.9.0 |
| 科学计算 | scipy | 1.18.0 |
| 数据处理 | pandas | 3.0.3 |
| 可视化 | matplotlib | 3.11.0 |
| 压缩支持 | lz4 / zstandard | 4.4.5 / 0.25.0 |
| 进度条 | tqdm | 4.68.3 |
