# 第7章 · 频谱分析与功率谱密度估计

---

## 0. 从时域到频域：为什么要学这一章？

### 0.1 你已经在用"频率思维"了

前六章你一直在和频率打交道——低通保留低频（趋势）、高通保留高频（波动）、带通提取特定周期。但你只是"在频率域提了要求"，还没有真正**看到**信号的频率成分长什么样。

打个比方。前六章你像是一位厨师，知道"加点糖会让菜变甜"、"加点盐会让菜变咸"，但你还没有尝过一道菜的"味道谱"——不知道它里面到底有多少甜、多少酸、多少苦。

频谱分析就是"品尝"你的信号——**它告诉你信号里每种频率各有多少分量。**

### 0.2 这章能回答什么问题？

你有一组日度数据（股价、温度、用电量……），直观上你看不出什么规律。频谱分析可以回答：

- "除了肉眼可见的波动，还有隐藏的周期性吗？" → **FFT 幅度谱**
- "哪个频率的波动占主导地位？" → **功率谱密度（PSD）**
- "我的数据不是等间隔采样的，还能分析频谱吗？" → **Lomb-Scargle**
- "两个信号在哪些频率上同步变化？" → **相干性（Coherence）**

### 0.3 和你的统计学背景对接

| 统计学概念 | 信号处理对应 |
|-----------|-------------|
| 方差 Var(X) | 信号的总功率（所有频率上的功率之和） |
| 把总方差分解为各因子贡献（ANOVA） | 把总功率分解为各频率成分贡献（频谱） |
| 自协方差函数 γ(h) | 功率谱 = 自协方差的傅里叶变换（Wiener-Khinchin 定理） |
| 两个变量的相关系数 ρ | 两个信号在某个频率上的相干性 |
| 样本均值的标准误（随 N 增大而减小） | Welch 方法（分段平均降低谱估计方差） |

## 场景一："我的数据里有隐藏的周期吗？"

### 1.1 问题

你拿到一组日度数据，比如一年的气温记录或者电力负荷。你看到夏天高、冬天低——这是年周期。但有没有更短的隐藏周期？比如每 3 天一个波动、每 7 天一个循环？肉眼看不出来。

### 1.2 傅里叶变换做什么（终极精简版）

傅里叶变换 = 把信号从"时间表示"换成"频率表示"。

- 输入：一组按时间排列的数值 $x[0], x[1], ..., x[N-1]$
- 输出：一组复数 $X[0], X[1], ..., X[N-1]$
- $X[k]$ 的含义：**信号中有多少"每 N 个样本完成 k 个周期"的正弦波成分**
- $|X[k]|$（复数的模）= 这个频率成分的强度（幅度）
- $\angle X[k]$（复数的角度）= 这个频率成分的起始位置（相位）

**统计学类比**：傅里叶变换就像 PCA——把数据从"原始坐标轴"变换到"主成分方向"，只不过 PCA 的方向是方差最大的方向，而傅里叶变换的方向是不同频率的正弦波。

### 1.3 频率轴怎么构建？

FFT 算完之后，$X[0]$ 对应频率 0（均值/DC），$X[1]$ 对应什么频率？$X[N-1]$ 对应什么频率？

```python
from scipy.fft import fft, fftfreq

X = fft(x)                    # 复数频谱
freqs = fftfreq(N, d=1/fs)   # 频率轴（Hz）
```

`fftfreq` 返回的频率排列是：`[0, 1, 2, ..., N/2-1, -N/2, ..., -1] × fs/N`。

- 前半部分（索引 0 到 N/2-1）：正频率，从小到大
- 后半部分（索引 N/2 到 N-1）：负频率，对于实信号和前半部分是对称的

**如果你处理的是实信号（99% 的情况），用 `rfft` 更方便**——它只返回正频率部分，省一半计算和存储：

```python
from scipy.fft import rfft, rfftfreq

X = rfft(x)                          # 只包含正频率
freqs = rfftfreq(N, d=1/fs)          # 对应的频率轴
```

$X[0]$ 对应频率 = 0（信号的 DC 分量，即均值 × N）。$X[k]$ 对应频率 = k × fs / N。最高频率 = fs / 2（Nyquist 频率）。

### 1.4 一个关键细节：频率分辨率

**频率分辨率 = fs / N。** 这是相邻两个频率"格子"之间的间距。

这意味着：如果你有 1000 个点，采样率 1000 Hz，频率分辨率 = 1 Hz。你能区分 10 Hz 和 11 Hz，但无法区分 10.3 Hz 和 10.7 Hz——它们会混在同一个"格子"里。

这是 FFT 的天生限制，不是 bug。要提高频率分辨率，唯一的方法是**增加信号长度**（增大 N）。

### 1.5 幅度谱的正确解读

```python
from scipy.fft import rfft, rfftfreq

X = rfft(x)
freqs = rfftfreq(N, d=1/fs)
amplitude = np.abs(X) / N      # 除以 N 得到实际的幅度值
amplitude[1:-1] *= 2           # 补偿单边谱（除了 DC 和 Nyquist）
```

为什么要除以 N？因为 FFT 定义里有一个隐含的缩放因子。不除以 N 的话，幅度值不是"原信号的幅度"，而是"幅度 × N"。

为什么要乘以 2？因为 `rfft` 只保留了正频率。正频率的能量应该等于总能量（对于实信号，正负频率对称各分一半），所以除了频率=0（DC）和频率=Nyquist 这两个特殊位置，都要乘以 2 来补偿。

## 场景二："我想估计信号的功率谱密度"

### 2.1 问题

FFT 幅度谱的直接做法有一个致命缺陷：**即便你对同一个随机过程重复采样，每次算出来的 FFT 幅度谱都可能大不相同。** 它非常"吵"——毛刺多、不稳定。

这是统计学中的经典问题：**点估计的方差太大。** 你需要一个更稳定的估计方法。

### 2.2 功率谱密度（PSD）是什么？

**一句话**：PSD 告诉你"单位频率区间内的平均功率是多少"。x 轴是频率，y 轴是功率密度（类似概率密度——在某个频率区间积分得到该区间的总功率）。

**和你的统计学知识对接**：
- 信号的总方差 = 所有频率的 PSD 之和（积分）× 频率分辨率
- 频率 f₁ 到 f₂ 之间的功率 = PSD 在 [f₁, f₂] 上的积分
- PSD 大 = 这个频率的波动在信号中占主导地位

### 2.3 `periodogram` — 最基础的 PSD 估计

```python
from scipy.signal import periodogram

freqs, psd = periodogram(x, fs=fs)
```

`periodogram` 就是 FFT 幅度谱的平方（加上适当缩放）。它继承了 FFT 的所有问题：**方差大、毛刺多、不可靠。**

### 2.4 Welch 方法 — 分段平均，降低方差

Welch 方法解决了周期图的方差问题。核心思路很简单，用统计学的话说就是"用多个子样本的均值代替单一样本估计"：

1. 把信号切成多段（可以重叠）
2. 对每一段单独做周期图
3. 把各段的周期图取平均

**统计学类比**：你估计总体均值时，与其用 10000 个点算一个样本均值（这其实已经是很好的估计了），不如把 10000 个点分成 10 段各 1000 个点，每段算一个均值，然后把这 10 个均值再平均。FFT 的周期图没有大数定律保护——取更多点不会让方差降低（因为周期图不是一致估计量）。Welch 方法的分段平均解决了这个问题。

```python
from scipy.signal import welch

freqs, psd = welch(x, fs=fs, nperseg=256, noverlap=128, window='hann')
```

三个关键参数：

**`nperseg` — 每段长度**。这是最重要的参数。nperseg 越大 → 频率分辨率越高（能区分更接近的频率），但分段数越少 → 方差降低得越少。nperseg 越小 → 方差越低（更平滑），但频率分辨率越低。**这是分辨率 vs 方差的经典权衡。**

**`noverlap` — 段间重叠点数**。让相邻段有重叠，可以在不牺牲频率分辨率的情况下增加分段数（因为一段的末尾数据可以同时出现在下一段的开头）。典型值 = `nperseg // 2`。

**`window` — 窗函数**。和第 3 章学的一样——段的两端要平滑衰减到零，避免截断造成的频谱泄漏。默认 `'hann'` 基本够用。

### 2.5 Welch 参数选择指南

| 你的需求 | nperseg | noverlap | 效果 |
|---------|---------|----------|------|
| 需要高频率分辨率（如区分 10.1Hz 和 10.3Hz） | 大（信号长度的 1/4~1/2） | nperseg/2 | 分辨率高，但曲线较"毛" |
| 需要光滑的谱估计（如只看大致的频带分布） | 小（信号长度的 1/10~1/16） | nperseg/2 | 光滑，但频率分辨率粗糙 |
| 两者都想要（通常的需求） | 信号长度的 1/4 ~ 1/8 | nperseg/2 | 折中 |

## 场景三："我的数据采样不均匀，还能做频谱分析吗？"

### 3.1 问题

标准 FFT 要求数据是等间隔采样的。但现实中你可能遇到：
- 金融数据：交易日之间有周末和假期，不是均匀的每天
- 天文观测：只能在某些夜晚观测到目标
- 临床数据：病人来门诊的时间不固定

这个问题不能通过"填 NaN 再插值"来简单解决——插值会引入虚假的频率成分。

### 3.2 `lombscargle` — 非均匀采样的频谱

```python
from scipy.signal import lombscargle

freqs = np.linspace(0.01, 0.5, 200)  # 要考察的频率范围
pgram = lombscargle(t, x, freqs, normalize=True)
```

Lomb-Scargle 方法的思想：对每个你关心的频率，用最小二乘法拟合一个该频率的正弦波，看拟合优度有多高。拟合优度越高 = 该频率越可能是真实存在的周期成分。它绕过了"数据必须等间隔"的限制。

**统计学直觉**：对每个候选频率 f，你做如下回归：
```
x[t] = β₀ + β₁·sin(2πft) + β₂·cos(2πft) + ε
```
然后计算这个回归的 R²。对所有 f 都做一遍，R² 高的那些 f 就是可能的周期。Lomb-Scargle 把这个过程做了高效实现。

## 场景四："两个信号在哪些频率上同步变化？"

### 4.1 问题

你有两个信号——比如两只股票的日收益率，或者两个区域的电力负荷。你想知道：
- 在 20-60 天的周期范围内，它们是否同涨同跌？
- 在 5 天以下的日间波动中，它们是否独立？

**时域相关（你学过的 Pearson 相关系数）只能回答"总体上是否相关"**——把所有频率混在一起算一个数。如果信号在低频高度同步但在高频完全不相关，一个总相关系数会被稀释掉，看不出来。

### 4.2 `coherence` — 分频率的"相关系数"

```python
from scipy.signal import coherence

freqs, coh = coherence(x, y, fs=fs, nperseg=256)
# coh 范围：0（完全不相关）到 1（完全相关）
```

相干性 = 两个信号在每个频率上的"相关系数"。它告诉你：两个信号在频率 f 附近是否同步波动。coh=1 意味着在频率 f 上两个信号完全同步（一个涨另一个一定涨），coh=0 意味着在频率 f 上完全独立。

**统计学类比**：全局相关系数 = 把所有频率的协方差加起来再归一化。相干性 = 在每个频率上分别算相关系数。就像 ANOVA 把总方差分解到不同因子——相干性把总相关性分解到不同频带。

### 4.3 伴随的 `csd` — 互功率谱密度

```python
from scipy.signal import csd

freqs, csd_xy = csd(x, y, fs=fs, nperseg=256)
```

CSD = 每个频率上两个信号的"协方差"（复数形式）。相干性本质上是归一化后的 CSD：coh = |CSD|² / (PSD_x × PSD_y)。

## 5. 窗函数在频谱分析中的角色

### 5.1 频谱泄漏

FFT 把你的有限长信号当作"周期性无限重复"的——它假设信号的末尾和开头是接在一起的。如果实际上不是这样（第 N-1 个点和第 0 个点不相等），FFT 会"看到"一个不存在的跳变，把能量泄漏到附近的频率格子里。

### 5.2 加窗减轻泄漏

在 FFT 之前乘一个窗函数（Hann、Hamming 等），让信号的两端平滑衰减到零。这样 FFT 就不会"看到"不存在的跳变了。代价是频率分辨率略微降低（主瓣变宽），但泄漏大幅减少。

**实战原则**：
- 除非你的信号恰好是整数个周期（极其罕见），否则**做 FFT 之前总是加窗**
- 默认用 `'hann'` 窗
- 如果你需要精确测量某个频率的幅度（不关心邻近频率），可以考虑 `'flattop'` 窗（它在主瓣内的响应非常平坦）

## 6. 本章要点速查

| 概念 | 一句话 |
|------|--------|
| FFT / 幅度谱 | 看信号里每种频率各有多少分量——但方差大、不稳定 |
| PSD（功率谱密度） | 单位频率的平均功率——比幅度谱更统计、更稳定 |
| Welch 方法 | 分段平均降低方差——**工业标准** |
| nperseg | Welch 的核心参数：大=高分辨率，小=低方差 |
| Lomb-Scargle | 非均匀采样数据的频谱——用最小二乘拟合绕过等间隔要求 |
| 相干性 (coherence) | 每个频率上的"相关系数"——两个信号在不同频带的同步程度 |
| CSD (互功率谱) | 每个频率上两个信号的协方差 |
| 频谱泄漏 | FFT 认为信号无限循环——端点不匹配的能量泄漏到邻频 |
| 加窗 | FFT 前让信号两端平滑归零——减少泄漏 |
