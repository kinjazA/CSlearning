## 一、快速生成测试信号

```python
import numpy as np
from scipy import signal
import matplotlib.pyplot as plt

fs = 100          # 采样率 (Hz)
t = np.arange(0, 5, 1/fs)   # 5 秒的数据
```

### 1.1 纯正弦波 / 多频组合

```python
# 单频
y = np.sin(2 * np.pi * 5 * t)                           # 5 Hz 正弦波

# 多频叠加 + 噪声
y = (np.sin(2 * np.pi * 2 * t)                           # 2 Hz 基频
   + 0.5 * np.sin(2 * np.pi * 10 * t)                    # 10 Hz 谐波
   + 0.3 * np.random.randn(len(t)))                      # 高斯噪声
```

### 1.2 扫频信号 (chirp) — 频率随时间变化

```python
# 线性扫频：从 1 Hz 扫到 20 Hz
y = signal.chirp(t, f0=1, f1=20, t1=5, method='linear')

# 对数扫频（低频停留更久）
y = signal.chirp(t, f0=1, f1=20, t1=5, method='logarithmic')
```

### 1.3 方波 / 锯齿波

```python
y_square = signal.square(2 * np.pi * 3 * t, duty=0.5)    # 3 Hz 方波, 占空比 50%
y_saw = signal.sawtooth(2 * np.pi * 3 * t, width=1)      # 3 Hz 锯齿波
```

### 1.4 脉冲信号

```python
# 高斯调制脉冲
y = signal.gausspulse(t - 2.5, fc=10, bw=0.5)            # 中心 2.5s, 载频 10Hz

# 单位脉冲（用于测试系统响应）
imp = signal.unit_impulse(100, idx=50)                    # 100 点, 脉冲在位置 50
```

### 1.5 快速看信号

```python
def quick_plot(y, t=None, title=''):
    """随手画信号 — 日常调试用"""
    if t is None:
        t = np.arange(len(y))
    plt.figure(figsize=(14, 3))
    plt.plot(t, y, linewidth=0.6)
    plt.title(title)
    plt.grid(True, alpha=0.3)
    plt.tight_layout()
    plt.show()
```

## 二、卷积与相关：平滑、匹配、时延

### 2.1 三种平滑核（比 rolling.mean 更灵活）

```python
from scipy.signal import convolve

N = 5  # 窗口长度

# 简单移动平均 SMA（等权重）
kernel_sma = np.ones(N) / N

# 线性加权 WMA（越近权重越大）
kernel_wma = np.arange(1, N+1) / np.arange(1, N+1).sum()

# 三角形加权（中间权重最大，更平滑）
kernel_tri = np.concatenate([np.arange(1, N//2+2), np.arange(N//2, 0, -1)])
kernel_tri = kernel_tri / kernel_tri.sum()

# 应用
smoothed = convolve(signal, kernel_sma, mode='same')
```

### 2.2 mode 选择速查

| mode | 输出长度 | 何时用 |
|------|---------|--------|
| `'same'` | = 输入长度 | **最常用** — 和原始时间轴对齐 |
| `'valid'` | N - M + 1 | 不要边界效应，接受序列变短 |
| `'full'` | N + M - 1 | 调试时看边界效应 |

### 2.3 核很长时用 fftconvolve

```python
from scipy.signal import fftconvolve

# 核长度 > 500 时 fftconvolve 更快
smoothed = fftconvolve(signal, long_kernel, mode='same')
```

### 2.4 差分核 — 计算变化

```python
# 一阶差分：今天减昨天
kernel_diff1 = np.array([-1, 1])
change = convolve(signal, kernel_diff1, mode='same')

# 二阶差分：变化的变化（检测趋势加速/衰竭）
kernel_diff2 = np.array([1, -2, 1])
accel = convolve(signal, kernel_diff2, mode='same')
```

### 2.5 互相关 — 模板匹配和 lead-lag 分析

```python
from scipy.signal import correlate

# 在信号中搜索模板 pattern
match_score = correlate(signal, pattern, mode='same')     # 得分峰值 = 匹配位置

# lead-lag 分析：B 滞后 A 多少？
corr_ab = correlate(returns_A, returns_B, mode='full')
lags = np.arange(-len(returns_A) + 1, len(returns_B))
best_lag = lags[np.argmax(corr_ab)]                       # B 滞后 A 的天数
```

### 2.6 自相关 — 等同于统计学的 ACF

```python
acf = correlate(signal, signal, mode='full')
acf = acf[len(acf)//2:]                                   # 只取 lag >= 0
acf = acf / acf[0]                                        # 归一化：ACF[0]=1
```

## 三、FIR 滤波器设计（窗口法）

### 3.1 firwin — 主力函数

```python
from scipy.signal import firwin, freqz

fs = 100  # 采样率

# ---- 低通：保留 < 10 Hz ----
b_lp = firwin(numtaps=51, cutoff=10, fs=fs, window='hann')

# ---- 高通：保留 > 5 Hz ----
b_hp = firwin(numtaps=51, cutoff=5, fs=fs, window='hann', pass_zero='highpass')

# ---- 带通：保留 8~15 Hz ----
b_bp = firwin(numtaps=51, cutoff=[8, 15], fs=fs, window='hann', pass_zero='bandpass')

# ---- 带阻：去掉 48~52 Hz（工频干扰）----
b_bs = firwin(numtaps=51, cutoff=[48, 52], fs=fs, window='hann', pass_zero='bandstop')
```

### 3.2 参数选择速查

| 参数 | 怎么选 |
|------|--------|
| `numtaps` | 奇数。越大 → 过渡带越窄 → 延迟越大。起始值：51~101 |
| `cutoff` | ≤ Nyquist（= fs/2）。传了 `fs` 就用实际频率单位 |
| `window` | 默认 `'hann'`。需要更干净阻带 → `'hamming'`。需要自定义权衡 → `('kaiser', beta)` |
| `pass_zero` | `True`=低通, `False`=高通, `'bandpass'`=带通, `'bandstop'`=带阻 |

### 3.3 Kaiser 窗参数速查

| β | 旁瓣水平 | 过渡带 | 相当于 |
|---|---------|--------|--------|
| 0 | -13 dB | 最窄 | 矩形窗（不推荐） |
| 3~4 | -25~-28 dB | 较窄 | 比 hann 稍快 |
| 5~6 | -36~-40 dB | 中等 | 接近 hann/hamming |
| 7~8 | -48~-53 dB | 较宽 | 阻带很干净 |
| 10+ | < -70 dB | 宽 | 极端干净的阻带 |

### 3.4 验证滤波器 — 必做步骤

```python
w, h = freqz(b_lp, worN=8000, fs=fs)          # 频率响应
magnitude_db = 20 * np.log10(np.abs(h))        # 转 dB

plt.figure(figsize=(12, 4))
plt.plot(w, magnitude_db)
plt.axhline(-3, color='gray', linestyle='--', alpha=0.5, label='-3 dB')
plt.xlabel('Frequency (Hz)'); plt.ylabel('Magnitude (dB)')
plt.title('Frequency Response'); plt.grid(True, alpha=0.3); plt.show()
```

## 四、FIR 滤波器设计（Remez 最优法）

### 4.1 remez — 等波纹最优设计

```python
from scipy.signal import remez

fs = 100

# bands: [通带起, 通带止, 阻带起, 阻带止] — 单位 Hz（传了 fs）
# desired: 对应每段的增益 [通带增益, 阻带增益]
# weight: 各段的误差权重 [通带权重, 阻带权重] — 越大越重视

# 低通：0~8 Hz 通过，12~50 Hz 阻挡
b = remez(numtaps=51,
          bands=[0, 8, 12, 50],         # 通带 [0,8] + 过渡带 [8,12] + 阻带 [12,50]
          desired=[1, 0],                # 通带增益=1, 阻带增益=0
          weight=[1, 10],                # 阻带误差权重是通带的 10 倍（更重视阻带干净）
          fs=fs)
```

### 4.2 remez vs firwin 选择

| 场景 | 用什么 |
|------|--------|
| 只关心一个截止频率，快速搞定 | `firwin` |
| 同等阶数下想要最窄过渡带 | `remez` |
| 需要多段频带有不同增益（如 1 → 0.5 → 0） | `remez` |
| 需要精确指定各频带的误差权重 | `remez` |

## 五、IIR 滤波器设计

### 5.1 五种原型速查

```python
from scipy.signal import butter, cheby1, cheby2, ellip, bessel

fs = 100
order = 4
cutoff = 10  # Hz（传了 fs 就用实际单位）

# Butterworth — 最平坦，默认首选
sos = butter(order, cutoff, btype='low', fs=fs, output='sos')

# Chebyshev I — 通带有纹波，过渡带比 Butterworth 窄
sos = cheby1(order, rp=1, Wn=cutoff, btype='low', fs=fs, output='sos')

# Chebyshev II — 阻带有纹波，通带平坦
sos = cheby2(order, rs=40, Wn=cutoff, btype='low', fs=fs, output='sos')

# Elliptic — 过渡带最窄，但通带和阻带都有纹波
sos = ellip(order, rp=1, rs=40, Wn=cutoff, btype='low', fs=fs, output='sos')

# Bessel — 相位最线性，但过渡带最宽
sos = bessel(order, cutoff, btype='low', fs=fs, output='sos')
```

### 5.2 选择决策树

```
需要线性相位（波形不能扭曲）？ → 用 FIR（firwin/remez），不要用 IIR
需要实时 / 低阶数？          → 用 IIR
  ├─ 通带平坦最重要           → Butterworth
  ├─ 过渡带要窄、可接受通带纹波 → Chebyshev I
  ├─ 阻带要干净               → Chebyshev II
  ├─ 过渡带要极窄             → Elliptic
  └─ 相位线性最重要（IIR中）  → Bessel
```

### 5.3 必须用 SOS 格式（不要用 ba 格式）

```python
# ✅ 正确：output='sos'
sos = butter(4, 10, btype='low', fs=100, output='sos')
filtered = signal.sosfilt(sos, data)

# ❌ 危险：output='ba'（高阶时数值不稳定）
# b, a = butter(8, 10, btype='low', fs=100, output='ba')  # order>4 就容易出问题
```

## 六、滤波器应用：怎么把滤波器"套"到数据上

### 6.1 四大应用函数对比

| 函数 | 方向 | 相位 | 延迟 | 何时用 |
|------|------|------|------|--------|
| `lfilter(b,a,x)` | 因果（只用过去） | 非线性（IIR） | 有延迟 | 实时流式处理 |
| `filtfilt(b,a,x)` | 正反各一次 | **零相位** | 无 | **离线分析首选** |
| `sosfilt(sos,x)` | 因果 | 非线性 | 有延迟 | IIR 滤波标准方式 |
| `sosfiltfilt(sos,x)` | 正反各一次 | **零相位** | 无 | IIR 离线零相位 |

### 6.2 典型用法

```python
from scipy.signal import lfilter, filtfilt, sosfilt, sosfiltfilt

# FIR 离线去噪（零相位，无延迟）
b = firwin(51, 10, fs=100)
y = filtfilt(b, [1], data)                              # FIR: b 是分子, a=[1]

# IIR 离线去噪（零相位，SOS 格式）
sos = butter(4, 10, fs=100, output='sos')
y = sosfiltfilt(sos, data)

# 实时流式（因果，有延迟）— 先初始化状态
zi = signal.sosfilt_zi(sos) * data[0]                   # 初始状态
y_stream, zf = signal.sosfilt(sos, data, zi=zi)         # 逐块处理
```

### 6.3 正确做法总结

```
离线分析（有完整数据）→ filtfilt / sosfiltfilt ← 零相位，你几乎总是要这个
实时流式（数据逐条进来）→ sosfilt + sosfilt_zi ← 必须用因果滤波
IIR 高阶（order >= 4） → 必须用 SOS 格式 ← ba 格式数值不稳定
```

## 七、频谱分析与功率谱密度估计

### 7.1 Welch 方法 — 标准 PSD 估计

```python
from scipy.signal import welch

freqs, psd = welch(signal, fs=fs, nperseg=256, noverlap=128)

# 参数选择
# nperseg: 每段长度。越长 → 频率分辨率越高 → 方差越大。典型值：len(signal)//4 ~ len(signal)//8
# noverlap: 段间重叠。通常 = nperseg // 2
# 频率分辨率 = fs / nperseg
```

### 7.2 快速看频谱

```python
freqs, psd = welch(signal, fs=fs, nperseg=256)
plt.figure(figsize=(12, 4))
plt.semilogy(freqs, psd)                                 # semilogy 看噪声底更好
plt.xlabel('Frequency (Hz)'); plt.ylabel('PSD')
plt.grid(True, alpha=0.3); plt.show()
```

### 7.3 Lomb-Scargle — 非均匀采样数据

```python
from scipy.signal import lombscargle

# 非均匀采样：时间点不规则（如交易日）
t_uneven = np.array([...])  # 不均匀的时间点
y_uneven = np.array([...])  # 对应的值

freqs = np.linspace(0.01, 0.5, 200)                      # 要搜索的频率
angular_freqs = 2 * np.pi * freqs

pgram = lombscargle(t_uneven, y_uneven, angular_freqs, normalize=True)
```

### 7.4 相干性 — 两个信号在各频率上的关联程度

```python
from scipy.signal import coherence

f_coh, coh = coherence(signal_A, signal_B, fs=fs, nperseg=256)

# coh 范围 [0, 1]。接近 1 = 该频率上两个信号高度相关
```

## 八、时频分析：STFT、小波、Hilbert

### 8.1 STFT / 频谱图 — 频率如何随时间变化

```python
from scipy.signal import spectrogram, stft

# 直接画频谱图
f, t_spec, Sxx = spectrogram(signal, fs=fs, nperseg=256, noverlap=192)
plt.pcolormesh(t_spec, f, 10 * np.log10(Sxx), shading='gouraud')
plt.ylabel('Frequency (Hz)'); plt.xlabel('Time (s)')
plt.colorbar(label='PSD (dB)')

# 返回复数 STFT（用于重建）
f_stft, t_stft, Zxx = stft(signal, fs=fs, nperseg=256, noverlap=192)

# 参数权衡：
# nperseg 大 → 频率分辨率好、时间分辨率差
# nperseg 小 → 时间分辨率好、频率分辨率差
# 无法同时完美（Heisenberg 不确定性）
```

### 8.2 CWT 小波变换 — 更好的时频分辨率

```python
from scipy.signal import cwt, ricker, morlet2

widths = np.arange(1, 64)                                # 尺度范围（大致对应频率）

# Ricker 小波（墨西哥帽）
cwtmatr = cwt(signal, ricker, widths)

# Morlet 小波（更适合振荡信号）
cwtmatr = cwt(signal, morlet2, widths, w=5)              # w=5 控制包络宽度

plt.imshow(cwtmatr, extent=[0, len(signal)/fs, 1, 64],
           cmap='PRGn', aspect='auto')
```

### 8.3 Hilbert 变换 — 瞬时幅度、相位、频率

```python
from scipy.signal import hilbert

analytic = hilbert(signal)                               # 解析信号（复数）
envelope = np.abs(analytic)                              # 瞬时包络
inst_phase = np.unwrap(np.angle(analytic))               # 瞬时相位（解缠绕）
inst_freq = np.diff(inst_phase) / (2.0 * np.pi) * fs     # 瞬时频率
```

## 九、峰值检测

### 9.1 find_peaks — 主力函数

```python
from scipy.signal import find_peaks

# 基础用法
peaks, properties = find_peaks(signal)

# 用 prominence 过滤 — 最重要的参数
peaks, props = find_peaks(signal, prominence=0.5)        # 只保留突出度 > 0.5 的峰

# 其他约束
peaks, _ = find_peaks(signal,
                      height=1.0,                        # 最低高度
                      distance=10,                       # 峰之间最少间隔（采样点）
                      width=(2, 20),                     # 峰宽范围
                      threshold=0.3)                     # 峰和邻域的最小垂直差
```

### 9.2 prominence 直觉理解

```
prominence = 峰的高度 - 包围它的最低"等高线"
一个"真正的峰"需要从峰顶下降到足够低才能爬到另一个更高的峰

prominence 大 → 这个峰"突出"、显著
prominence 小 → 可能是噪声造成的假峰
```

### 9.3 峰属性提取

```python
peaks, props = find_peaks(signal, prominence=0.5)

props['prominences']         # 每个峰的突出度
props['widths']               # 每个峰的宽度
props['left_bases']           # 每个峰的左基座索引
props['right_bases']          # 每个峰的右基座索引
```

### 9.4 用包络找大趋势的峰

```python
# 先取包络（Hilbert），再在包络上找峰 — 找"事件"而非"振荡"
analytic = hilbert(signal)
envelope = np.abs(analytic)
events, _ = find_peaks(envelope, prominence=np.std(signal), distance=fs*0.5)
```

## 十、LTI 系统分析

### 10.1 从滤波器系数构建系统

```python
from scipy.signal import lti, freqz, impulse, step

# 从 FIR / IIR 系数构建 LTI 系统
b = firwin(51, 10, fs=100)
sys = lti(b, [1])                                        # FIR: 分母=1

sos = butter(4, 10, fs=100, output='sos')
b_all, a_all = signal.sos2tf(sos)
sys = lti(b_all, a_all)

# 从零极点构建
sys = lti(zeros=[-1+1j, -1-1j], poles=[-0.5+0.5j, -0.5-0.5j], gain=1.0)
```

### 10.2 系统分析

```python
# 频率响应 — 和 freqz 等价
w, mag, phase = sys.bode()                                # Bode 图

# 脉冲响应 — 系统对"瞬间刺激"的反应
t, y_imp = impulse(sys, T=np.linspace(0, 1, 100))

# 阶跃响应 — 系统对"突然打开"的反应
t, y_step = step(sys, T=np.linspace(0, 1, 100))
```

### 10.3 零极点分析

```python
from scipy.signal import tf2zpk

z, p, k = tf2zpk(b, a)                                   # 零点、极点、增益

# 稳定性判据：所有极点的模 < 1（离散系统）
is_stable = np.all(np.abs(p) < 1)

# 画零极点图
plt.figure(figsize=(6, 6))
plt.scatter(np.real(z), np.imag(z), marker='o', s=40, label='Zeros')
plt.scatter(np.real(p), np.imag(p), marker='x', s=40, label='Poles')
theta = np.linspace(0, 2*np.pi, 200)
plt.plot(np.cos(theta), np.sin(theta), 'k--', alpha=0.3)   # 单位圆
plt.axhline(0, color='gray', alpha=0.3); plt.axvline(0, color='gray', alpha=0.3)
plt.legend(); plt.axis('equal'); plt.grid(True, alpha=0.3)
```

## 十一、去噪方法

### 11.1 方法选择速查

| 噪声类型 | 用什么 | 代码 |
|---------|--------|------|
| 持续高斯噪声，要保边缘 | Savitzky-Golay | `savgol_filter(x, window_length=11, polyorder=3)` |
| 偶发尖峰脉冲 | 中值滤波 | `medfilt(x, kernel_size=5)` |
| 平稳高斯噪声（自动适应） | Wiener | `wiener(x, mysize=11)` |
| 精确频率控制 | 低通 FIR/IIR | `filtfilt(firwin(...), [1], x)` |
| 固定频率干扰（如 50Hz） | 陷波器 | `firwin(51, [48,52], fs=fs, pass_zero='bandstop')` |
| 基线漂移 | 去趋势 | `detrend(x, type='linear')` |

### 11.2 各方法代码

```python
from scipy.signal import savgol_filter, medfilt, wiener, detrend

y_sg     = savgol_filter(x, window_length=11, polyorder=3)  # S-G 平滑
y_med    = medfilt(x, kernel_size=5)                        # 中值滤波（核为奇数）
y_wien   = wiener(x, mysize=11)                             # Wiener 自适应
y_det    = detrend(x, type='linear')                        # 去线性趋势

# 串联使用：先去尖峰，再去高斯噪声
y_clean  = wiener(medfilt(x, kernel_size=5), mysize=11)
```

### 11.3 Savitzky-Golay 参数速查

| window_length | polyorder | 效果 |
|:---:|:---:|------|
| 小 (5~7) | 低 (2~3) | 轻度平滑，保留大部分细节 |
| 中 (11~15) | 低 (2~3) | **默认起点** — 中等平滑，保边缘 |
| 中 (11~15) | 高 (4~5) | 追踪弯曲更多，保留噪声也更多 |
| 大 (21+) | 低 (2~3) | 强平滑，可能抹平窄特征 |

### 12.2 预处理管线模板

```python
from scipy.signal import detrend, decimate

class SignalPreprocessor:
    """链式 API — 每一步返回 self 以便链式调用"""
    def __init__(self, fs):
        self.fs = fs
        self.steps = []

    def detrend(self, type='linear'):
        self.steps.append(('detrend', type)); return self

    def lowpass(self, cutoff, numtaps=51):
        self.steps.append(('lowpass', cutoff, numtaps)); return self

    def decimate(self, q):
        self.steps.append(('decimate', q)); return self

    def normalize(self, method='zscore'):
        self.steps.append(('normalize', method)); return self

    def process(self, x):
        for step in self.steps:
            if step[0] == 'detrend':
                x = detrend(x, type=step[1])
            elif step[0] == 'lowpass':
                b = firwin(step[2], step[1], fs=self.fs)
                x = filtfilt(b, [1], x)
            elif step[0] == 'decimate':
                x = decimate(x, q=step[1])
                self.fs /= step[1]          # 更新采样率！
            elif step[0] == 'normalize':
                if step[1] == 'zscore':
                    x = (x - np.mean(x)) / np.std(x)
                elif step[1] == 'minmax':
                    x = (x - np.min(x)) / (np.max(x) - np.min(x))
        return x

# 用法：链式调用
preprocessor = (SignalPreprocessor(fs=100)
                .detrend('linear')
                .lowpass(cutoff=20, numtaps=51)
                .decimate(q=4)
                .normalize('zscore'))
clean = preprocessor.process(raw_data)
```

### 12.3 标准化方法

```python
zscore   = (x - np.mean(x)) / np.std(x)               # 均0方1 — 最通用
minmax   = (x - np.min(x)) / (np.max(x) - np.min(x))   # [0,1] — 保留比例
rms_norm = x / np.sqrt(np.mean(x**2))                  # 能量归一化
```

## 十三、信号特征工程（送入 ML）

### 13.1 单个窗口的特征提取函数

```python
from scipy.signal import welch, find_peaks, hilbert

def extract_signal_features(window, fs):
    """从一段信号窗口提取标准特征集。返回 dict。"""
    feat = {}
    N = len(window)
    sigma = np.std(window)

    # ---- 时域 ----
    feat['mean'] = np.mean(window)
    feat['std'] = sigma
    feat['rms'] = np.sqrt(np.mean(window**2))
    feat['skewness'] = np.mean((window - feat['mean'])**3) / (sigma**3) if sigma > 1e-10 else 0
    feat['kurtosis'] = np.mean((window - feat['mean'])**4) / (sigma**4) if sigma > 1e-10 else 0
    feat['crest_factor'] = np.max(np.abs(window)) / feat['rms'] if feat['rms'] > 0 else 0

    # ---- 频域 ----
    freqs, psd = welch(window, fs=fs, nperseg=min(64, N//4))
    total_power = np.sum(psd)
    if total_power > 1e-15:
        feat['spectral_centroid'] = np.sum(freqs * psd) / total_power
        feat['dominant_freq'] = freqs[np.argmax(psd)]
        psd_norm = psd / total_power
        feat['spectral_entropy'] = -np.sum(psd_norm[psd_norm > 0] * np.log(psd_norm[psd_norm > 0]))
        low_mask = freqs <= 2
        feat['low_freq_ratio'] = np.sum(psd[low_mask]) / total_power
    else:
        feat['spectral_centroid'] = 0; feat['dominant_freq'] = 0
        feat['spectral_entropy'] = 0; feat['low_freq_ratio'] = 0

    # ---- 峰特征 ----
    peaks, props = find_peaks(window, prominence=max(0.1, sigma*0.5))
    feat['n_peaks'] = len(peaks)
    feat['mean_prominence'] = np.mean(props['prominences']) if len(peaks) > 0 else 0

    # ---- 包络特征 ----
    envelope = np.abs(hilbert(window))
    feat['envelope_mean'] = np.mean(envelope)
    feat['envelope_std'] = np.std(envelope)

    return feat
```

### 13.2 滑动窗口特征提取 — 生成 ML 特征矩阵

```python
def sliding_window_features(signals, labels, window_size=256, stride=128, fs=1.0):
    """对多条信号做滑动窗口特征提取。返回 X (特征矩阵), y (标签)。"""
    X_list, y_list = [], []
    for sig, label in zip(signals, labels):
        for start in range(0, len(sig) - window_size, stride):
            window = sig[start:start + window_size]
            feats = extract_signal_features(window, fs)
            X_list.append(list(feats.values()))
            y_list.append(label)
    return np.array(X_list), np.array(y_list)
```

### 13.3 sklearn Transformer 封装 — 嵌入 Pipeline

```python
from sklearn.base import BaseEstimator, TransformerMixin

class SignalFeatureExtractor(BaseEstimator, TransformerMixin):
    """将滑动窗口特征提取做成 sklearn Transformer"""
    def __init__(self, window_size=256, stride=128, fs=1.0):
        self.window_size = window_size
        self.stride = stride
        self.fs = fs

    def fit(self, X, y=None):
        return self

    def transform(self, X):
        features_list = []
        for sig in X:
            for start in range(0, len(sig) - self.window_size, self.stride):
                window = sig[start:start + self.window_size]
                feats = extract_signal_features(window, self.fs)
                features_list.append(list(feats.values()))
        return np.array(features_list)

# 完整 Pipeline
from sklearn.pipeline import Pipeline
from sklearn.ensemble import RandomForestClassifier

pipeline = Pipeline([
    ('features', SignalFeatureExtractor(window_size=256, stride=128, fs=100)),
    ('scaler', StandardScaler()),
    ('clf', RandomForestClassifier(n_estimators=100))
])
pipeline.fit(X_train_signals, y_train)
```

## 十五、常见坑与排错清单

### 坑 1：没传 `fs` 参数
```python
# ❌ 错：cutoff=10, 没传 fs → scipy 默认 Nyquist=1 → 10 Hz > 1 Hz → 报错或结果全错
b = firwin(51, cutoff=10)

# ✅ 正确
b = firwin(51, cutoff=10, fs=100)
```

### 坑 2：截止频率 > Nyquist
```python
fs = 100    # Nyquist = 50 Hz
# ❌ cutoff=60 > 50 → 报错
b = firwin(51, cutoff=60, fs=100)

# ✅ cutoff 必须 ≤ fs/2
b = firwin(51, cutoff=40, fs=100)
```

### 坑 3：IIR 高阶用 ba 格式
```python
# ❌ order=8 用 ba → 数值不稳定
b, a = butter(8, 10, fs=100)
filtered = lfilter(b, a, data)

# ✅ order≥4 必须用 SOS
sos = butter(8, 10, fs=100, output='sos')
filtered = sosfilt(sos, data)
```

### 坑 4：降采样前忘抗混叠
```python
# ❌ 直接降采样，高频折叠进低频
y_bad = x[::4]

# ✅ 用 decimate（自动抗混叠）或先低通滤波再降采样
y_good = decimate(x, q=4)
```

### 坑 5：numtaps 是偶数
```python
# ❌ FIR 窗函数法要求奇数
b = firwin(50, cutoff=10, fs=100)    # 可能报错或相位异常

# ✅
b = firwin(51, cutoff=10, fs=100)
```

### 坑 6：定频出现的"假周期"可能是混叠
- 日度数据中周期 < 2 天的所有波动都不可见，且可能混叠为更长的假周期
- 高频数据（如 1 分钟 tick）要检查你感兴趣的频率是否超过 Nyquist
- 规律：频率 f 会折叠到 |f - k×fs|（k 为整数）

### 坑 7：频谱泄漏
- Welch 的 `nperseg` 太小 → 频率分辨率不够 → 相邻的峰被抹平
- Welch 的 `nperseg` 太大 → 方差大 → PSD 曲线太嘈杂
- 推荐：`nperseg = len(signal) // 4` 到 `len(signal) // 8`，配合 `noverlap = nperseg // 2`

### 坑 8：实时处理用了零相位滤波
```python
# ❌ filtfilt 需要未来数据，实时场景不可能用
y = filtfilt(b, [1], streaming_data)     # 从概念上就错了

# ✅ 实时用 lfilter / sosfilt + 初始状态
zi = signal.sosfilt_zi(sos) * data[0]
y, zf = sosfilt(sos, data, zi=zi)
```
