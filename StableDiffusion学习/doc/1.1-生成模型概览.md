# 1.1 生成模型概览

## 什么是生成模型

深度学习最引人入胜的能力之一，是它不仅能**判别**——判断这张图是猫还是狗，这段话是正面还是负面情绪——它还能**创造**。生成模型的目标，是学习一个数据分布 $p_{\text{data}}(x)$，然后从这个分布中采样出全新的、逼真的样本。这里的 $x$ 可以是图像、文本、音频、视频，甚至蛋白质结构。在 Stable Diffusion 的语境下，我们关心的 $x$ 就是一张图像——可能是一幅 512×512 的 RGB 图片，展开后有近 80 万个维度的高维空间中的一点。

学习高维数据的分布本身是一件困难的事。我们手头通常只有一堆真实样本 $$x_1, x_2, \dots, x_N \sim p_{\text{data}}(x)$$，却不知道 $p_{\text{data}}(x)$ 的确切数学形式。生成模型要做的，就是用某种方式逼近这个分布，让我们得以从中采样。

在扩散模型成为主流之前，有三种范式统治着生成式 AI 的版图。

## GAN：对抗博弈

生成对抗网络（Generative Adversarial Network, GAN）将生成问题建模为一场博弈。系统由两方组成：一个**生成器** $G(z)$，它从一个随机噪声向量 $z \sim \mathcal{N}(0, I)$ 出发，试图"伪造"出逼真的样本；一个**判别器** $D(x)$，它试图区分真实样本和生成器产出的假样本。生成器想要骗过判别器，判别器想要看穿生成器——这是一个极小极大的对抗游戏：

$$\min_G \max_D \ \mathbb{E}_{x \sim p_{\text{data}}}[\log D(x)] + \mathbb{E}_{z \sim p(z)}[\log(1 - D(G(z)))]$$

GAN 的思想极其精巧，它在训练上的优势也很明显：推理时只需要一次前向传播就能生成一张图，速度很快。StyleGAN 系列曾经在逼真人脸生成上达到了惊艳的水平。

然而，GAN 的痛点也同样深刻。训练过程不稳定是它最臭名昭著的问题——生成器和判别器必须小心翼翼地保持平衡，一方太强另一方就学不到东西，导致模式崩溃（Mode Collapse），即生成器只会产出少数几种"安全"的样本，丧失了多样性。此外，GAN 的隐空间控制和条件注入也比较粗糙，不太容易精确地引导生成内容。

## VAE：隐空间为桥

变分自编码器（Variational Autoencoder, VAE）则走了一条完全不同的路。它不玩博弈，而是引入了隐变量 $z$ 作为中介，试图最大化数据的似然：

$$\log p(x) \geq \mathbb{E}_{q_\phi(z|x)}[\log p_\theta(x|z)] - D_{\text{KL}}(q_\phi(z|x) \parallel p(z))$$

这个式子叫 ELBO（Evidence Lower Bound）。直观上，第一项是**重建损失**：给定一个隐变量能多好地恢复出原图；第二项是**KL 散度正则化**：让学到的隐分布尽量靠近一个标准高斯先验。编码器 $q_\phi(z|x)$ 将图像压缩为隐变量，解码器 $p_\theta(x|z)$ 再从隐变量重建图像。

VAE 天然具有一个光滑、结构良好的隐空间，因为它强制性地把图像嵌入到一个高斯分布框架中，使得相近的隐向量在解码后得到的图像也相近，这意味着我们可以在隐空间中做插值。这个思路后来被 Stable Diffusion 继承——但不是直接用 VAE 生成图像，而是用 VAE 把图像压缩到一个**潜空间**（latent space），在潜空间中做扩散。

VAE 的致命弱点是模糊。KL 正则化虽然保证了隐空间的规整性，却也限制了模型捕捉高频细节的能力。VAE 生成的图像通常缺少纹理和锐利的边缘，在高分辨率自然图像上远不如 GAN 看起来真实。

## Autoregressive：逐像素作画

自回归模型把图像生成看作一个序列预测任务。它将一张图片拆成像素序列，然后一个像素接一个像素地生成：

$$p(x) = \prod_{i=1}^{n} p(x_i \mid x_1, x_2, \dots, x_{i-1})$$

PixelCNN 是早期代表，GPT 类模型把 Transformer 架构引入后催生了 ImageGPT 和 DALL·E 1。这类方法的优势在于训练稳定（就是标准的极大似然估计），不会像 GAN 那样模式崩溃，也天然适合跟语言模型结合——文本到图像的生成可以组织成"先看文本，再一个 token 一个 token 地解码图像"的范式。

但缺点也很明显：推理极慢。一张 1024×1024 的图有上百万个像素（即使在压缩的 token 空间中仍有成千上万个 token），必须逐个生成，无法并行。而且这种从左到右、从上到下的生成顺序，对应的是一种人为的因果偏序，而图像本身并没有天然的顺序——这是自回归模型在图像生成上的根本性结构不适配。

## Diffusion：退一步，海阔天空

扩散模型的思想可以用一句话概括：**与其一次性直接从噪声跳到一张完整的图，不如一小步一小步地走**。

具体地说，扩散模型的运作分为两个阶段。第一步是**前向过程**，我们在一张干净的图像上逐步添加高斯噪声，总共加 $T$ 步（通常 $T=1000$），直到图像完全变成一个各向同性的纯高斯噪声。这个过程不涉及学习——它是固定的，每一步只是简单地做：

$$x_t = \sqrt{1 - \beta_t} \ x_{t-1} + \sqrt{\beta_t} \ \epsilon, \quad \epsilon \sim \mathcal{N}(0, I)$$

第二步是**反向过程**，也即生成过程。我们训练一个神经网络——在 Stable Diffusion 中是 U-Net——来学习如何逆转每一步的加噪操作：给定一个被噪声污染的图像 $x_t$（以及当前的时刻 $t$），预测出其中混入了多少噪声，然后把它扣除。经过 $T$ 步的反复去噪，一个纯噪声最终会蜕变出一张逼真的图像。

为什么这个方案比前几种更好？

首先，**训练极其稳定**。扩散模型的训练目标本质上是一个简单的回归任务——预测添加的噪声，没有对抗训练的不稳定性，没有 VAE 中 KL 坍塌的隐患。损失函数就是预测噪声和真实噪声之间的 MSE：$$\mathcal{L} = \mathbb{E}_{x_0, \epsilon, t} \left[ \|\epsilon - \epsilon_\theta(x_t, t)\|^2 \right]$$

其次，**质量与多样性兼得**。DDPM 的作者在论文中展示了扩散模型在 FID 指标上超越当时最好的 GAN 的结果，而且不会模式崩溃——因为每一步去噪都是从条件高斯分布中采样，天然的随机性杜绝了单一种类的输出。

最后，**条件注入极其灵活**。你可以在去噪的每一步中，通过 Cross-Attention 把文本、图像、布局、深度图等信息注入网络，而不需要改变网络的基本结构。这让扩散模型天然适合做可控生成——而这一点，正是 Stable Diffusion 和 ControlNet 等后续工作大放异彩的基础。

## 从 DDPM 到 Stable Diffusion

DDPM（Denoising Diffusion Probabilistic Models, 2020）在像素空间上做扩散，这是它的主要瓶颈。一张 512×512 的图像意味着模型要在 262,144 维的空间中反复采样——即使 $T=50$ 步（DDIM 加速后），也意味着 50 次完整的前馈推理，计算成本高昂。

Stable Diffusion 的关键创新把这一切推进了一大步：**在潜空间中做扩散**。先用 VAE 把图像压缩到一个低维潜空间（例如从 512×512×3 压缩到 64×64×4，仅 16,384 维），在潜空间中做所有扩散操作，最后再用 VAE 解码器将去噪后的潜变量还原为像素。维度的指数降低使得在消费级 GPU 上运行扩散模型成为现实。此外，Stable Diffusion 还引入了 CLIP 文本编码器通过 Cross-Attention 向 U-Net 注入语言语义，实现了文本到图像生成的突破性效果。

下一节，我们将从 DDPM 的前向扩散过程入手，逐步推导扩散模型的数学细节。
