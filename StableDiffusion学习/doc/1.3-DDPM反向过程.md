# 1.3 DDPM 反向过程

## 时间逆流

前向过程把一张图像逐渐变成噪声。这个方向是确定且廉价的——只需要高斯噪声和一个方差调度，不需要任何神经网络。真正困难的问题是反过来：给定一团纯噪声，如何一步步地"去噪"，最终还原出一张有意义的图像？这就是反向过程。它是生成的本质。

## 反向过程的理想形式

如果前向过程是一条马尔可夫链，那么反向过程在数学上是同一条链条的逆——同样是一条马尔可夫链，只不过方向反过来：

$$p_\theta(x_{0:T}) = p(x_T) \prod_{t=1}^{T} p_\theta(x_{t-1} \mid x_t)$$

起点 $p(x_T) = \mathcal{N}(0, I)$ 是纯噪声。中间的每一步 $p_\theta(x_{t-1} \mid x_t)$ 也是一个高斯分布，我们把它参数化为：

$$p_\theta(x_{t-1} \mid x_t) = \mathcal{N}\left(x_{t-1};\ \mu_\theta(x_t, t),\ \sigma_t^2 I\right)$$

这里 $\mu_\theta(x_t, t)$ 是一个由神经网络学习的均值函数，$\sigma_t^2$ 是方差（在 DDPM 原始设定中方差是固定的，不学习）。

关键问题在于：**真值 $q(x_{t-1} \mid x_t)$ 长什么样？** 如果我们知道它，就可以用神经网络 $\mu_\theta$ 去逼近它。

## 真值：条件化在 $x_0$ 上

$q(x_{t-1} \mid x_t)$ 直接求解非常困难——它需要对所有可能的 $x_0$ 做积分。但如果我们额外知道 $x_0$，情况就完全不同了：$q(x_{t-1} \mid x_t, x_0)$ 有漂亮的闭式解。

这是直觉上说得通的：如果你既知道当前的噪声版本 $x_t$，又知道原始的干净图像 $x_0$，那你完全可以算出从 $x_{t-1}$ 变到 $x_t$ 这中间究竟加了多少噪声——进而逆推出 $x_{t-1}$。

利用贝叶斯公式：$$q(x_{t-1} \mid x_t, x_0) = \frac{q(x_t \mid x_{t-1}, x_0)\, q(x_{t-1} \mid x_0)}{q(x_t \mid x_0)}$$

由于前向过程是马尔可夫的，$q(x_t \mid x_{t-1}, x_0) = q(x_t \mid x_{t-1})$。上式中的三项我们全部已知：

- $q(x_t \mid x_{t-1})$ 是单步前向：均值 $\sqrt{\alpha_t}\, x_{t-1}$，方差 $\beta_t I$
- $q(x_{t-1} \mid x_0)$ 是一步到位公式：均值 $\sqrt{\bar{\alpha}_{t-1}}\, x_0$，方差 $(1 - \bar{\alpha}_{t-1})I$
- $q(x_t \mid x_0)$ 也来自一步到位：均值 $\sqrt{\bar{\alpha}_t}\, x_0$，方差 $(1 - \bar{\alpha}_t)I$

把三个高斯分布的概率密度函数代入贝叶斯公式，经过若干页的代数运算（这里不展开一步一步的推导，核心是高斯的乘积仍是高斯），最终得出 $q(x_{t-1} \mid x_t, x_0)$ 也是一个高斯分布，其均值为：$$\tilde{\mu}_t(x_t, x_0) = \frac{\sqrt{\bar{\alpha}_{t-1}}\beta_t}{1 - \bar{\alpha}_t} x_0 + \frac{\sqrt{\alpha_t}(1 - \bar{\alpha}_{t-1})}{1 - \bar{\alpha}_t} x_t$$

方差为：$$\tilde{\beta}_t = \frac{1 - \bar{\alpha}_{t-1}}{1 - \bar{\alpha}_t} \beta_t$$

在 DDPM 中，方差 $\tilde{\beta}_t$ 被设为固定值 $\sigma_t^2 = \beta_t$ 或 $\tilde{\beta}_t$（两者在 $T=1000$ 时差异很小），不由网络学习。所以神经网络唯一的任务就是**逼近均值** $\tilde{\mu}_t$。

## 换一个预测目标：从 $x_0$ 到 $\epsilon$

直接让网络去预测 $\tilde{\mu}_t(x_t, x_0)$ 是一个可行的选择。但 DDPM 的作者发现，换一种参数化方式效果会好得多。

将 $x_t = \sqrt{\bar{\alpha}_t}\, x_0 + \sqrt{1 - \bar{\alpha}_t}\, \epsilon$ 变形为 $x_0 = \frac{1}{\sqrt{\bar{\alpha}_t}}(x_t - \sqrt{1 - \bar{\alpha}_t}\, \epsilon)$，代入 $\tilde{\mu}_t$ 的表达式中，可以把均值重写为只用 $x_t$ 和真实噪声 $\epsilon$ 表示的形式：$$\tilde{\mu}_t(x_t, x_0) = \frac{1}{\sqrt{\alpha_t}} \left(x_t - \frac{\beta_t}{\sqrt{1 - \bar{\alpha}_t}}\, \epsilon\right)$$

这意味着如果神经网络知道混合在 $x_t$ 里的噪声 $\epsilon$ 是什么，它就能算出正确的均值。因此，我们不需要让网络输出均值 $\mu_\theta$，而是让它输出一个噪声预测 $\epsilon_\theta(x_t, t)$，然后用同样形式的公式计算均值：$$\mu_\theta(x_t, t) = \frac{1}{\sqrt{\alpha_t}} \left(x_t - \frac{\beta_t}{\sqrt{1 - \bar{\alpha}_t}}\, \epsilon_\theta(x_t, t)\right)$$

这个参数化的优势在实践中被充分验证——预测噪声比直接预测均值或预测 $x_0$ 都更容易学，因为它更"对称"：输入是一个带噪图像，输出是一个同维度的噪声，网络不需要理解图像的高级语义就能做好局部预测。

## 简化损失函数

DDPM 的完整变分下界损失包含若干项，但作者经过实验简化后证明，以下这个简单到近乎粗暴的损失函数就能训练出优秀的扩散模型：$$\mathcal{L}_{\text{simple}} = \mathbb{E}_{t, x_0, \epsilon}\left[ \|\epsilon - \epsilon_\theta(x_t, t)\|^2 \right]$$

其中 $t$ 从 $\{1, 2, \dots, T\}$ 中均匀随机采样，$x_0$ 从训练集中采样，$\epsilon \sim \mathcal{N}(0, I)$，$x_t$ 用前向一步公式计算：$x_t = \sqrt{\bar{\alpha}_t}\, x_0 + \sqrt{1 - \bar{\alpha}_t}\, \epsilon$。

这个损失就是最简单的均方误差——预测的噪声和实际添加的噪声之间的 L2 距离。没有对抗损失，没有感知损失，没有 KL 散度，就是 MSE。它的工作方式是：对于每一个训练样本，随机选一个时间步 $t$，往原图上混入一定比例的噪声，然后让网络猜出混进去的是什么噪声。网络被迫学习在每一个噪声级别上的去噪能力——从接近干净的微小修正，到几乎全噪的结构重建。

## 训练算法

把上面的内容组合起来，DDPM 的训练流程非常干净：

1. 从训练集采样一张干净图像 $x_0$
2. 随机采样一个时间步 $t \sim \text{Uniform}(1, T)$
3. 采样一个标准高斯噪声 $\epsilon \sim \mathcal{N}(0, I)$
4. 构造带噪图像 $x_t = \sqrt{\bar{\alpha}_t}\, x_0 + \sqrt{1 - \bar{\alpha}_t}\, \epsilon$
5. 计算网络预测 $\epsilon_\theta(x_t, t)$ 和真实噪声 $\epsilon$ 之间的 MSE
6. 梯度下降，更新参数 $\theta$

这个算法的优雅之处在于每一步随机采样一个 $t$，而非遍历所有 $T$ 步——训练效率极高，每一步都有直接的监督信号。网络在大量随机时间步的训练中，逐渐学会了在任何噪声水平下去噪。

## 采样（生成）算法

训练完成后，生成一张新图像的过程就是反向扩散：

1. 从标准高斯分布采样 $x_T \sim \mathcal{N}(0, I)$
2. 对 $t = T, T-1, \dots, 1$ 依次执行：
   - 如果 $t > 1$，采样 $z \sim \mathcal{N}(0, I)$；如果 $t = 1$，令 $z = 0$
   - 用网络预测噪声 $\epsilon_\theta(x_t, t)$
   - 计算去噪一步：
     $$x_{t-1} = \frac{1}{\sqrt{\alpha_t}} \left(x_t - \frac{\beta_t}{\sqrt{1 - \bar{\alpha}_t}}\, \epsilon_\theta(x_t, t)\right) + \sigma_t z$$
3. 最终的 $x_0$ 即为生成图像

注意其中的随机项 $\sigma_t z$——这是扩散模型多样性的来源。即使在相同初始噪声 $x_T$ 下，每次采样中随机变量 $z$ 的不同实现也会导致不同的生成结果。这也是为什么扩散模型不会像 GAN 那样模式崩溃：采样中的内在随机性天然保证了多样性的下限。

## 总结：付出与回报

反向过程的学习本质上是一件非常朴素的事：往图里加噪声，然后让网络猜加了多少。所有复杂的数学——马尔可夫链、高斯贝叶斯推导、变分下界——最终收敛于一个极其简单的训练损失 $\|\epsilon - \epsilon_\theta(x_t, t)\|^2$。

这种"复杂的理论 ⇒ 简单的算法"的模式，是扩散模型最打动人的特质之一。你不需要理解贝叶斯公式的每一个代数变换来使用扩散模型，但如果你理解了，你就会明白为什么这个简单的 MSE 损失是严格推导的必然结果，而非随手一试的经验巧合。
