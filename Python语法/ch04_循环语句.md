## 01 while循环

> * 判断条件是布尔类型，可以用比较运算，**True表示继续循环，False表示结束循环**
> * 需要设置循环终止的条件，否则将无限循环下去(通常就是`i += 1`这样的递增到临界值再出循环)

```
while 判断条件：
	做的事情1
	做的事情2
	……
```
```python
i = 0
while i < 100:
	print("111")
	i += 1 # 这个递增的索引是为了使while判断条件为Fasle结束循环
```
```python
# 求1~100的和
i = 0
sum = 0
while i < 101:
	sum += i
	i += 1
print(f"1~100的累加和是{sum}")
```
```python
# 随机选一个100以内的数字，不限次数的猜测，并记录猜了几次
import random
num = random.randint(1, 100)
count = 0
flag = True
while flag:
    guess_num = int(input("请输入你猜测的数字："))
    count += 1 #猜一次增加1，用作计数
    if guess_num == num:
        print("猜对了")
        flag = False # 终止循环的条件
    else:
        if guess_num < num:
            print("小了")
        else:
            print("大了")
print(f"你一共猜测了{count}次") 
```

## 02 while循环嵌套

```python
"""
例子：小明连续一百天向小美每天发一条表白信息，每次表白前又会连续送10朵玫瑰花
"""
i = 1
while i <= 100:
	print(f"今天是第{i}天，准备表白……")
	# 内层循环
	j = 1
	while j <= 10:
		print(f"这是送给小美的第{j}支玫瑰花")
		j += 1
	print("小美，我喜欢你")
	i += 1
print(f"坚持到第{i - 1}天，表白成功")
```

## 05 for循环

> * **for循环的循环条件是自定义的**，自行控制循环条件，是一种”轮询“机制，是对一批内容进行逐个处理,也称遍历循环
> * 有 in 和 not in两个操作符可以用于布尔值判断
```
for 临时变量 in 待处理数据集（序列）：
	循环满足条件时执行的代码
```
```python
# 将name里的内容挨个输出
name = "ilovepython"
for x in name:
	print(x)
```
```python
# 判断字符串中某个元素的出现次数
str = "asdfghjklaaagergaaa"
num_a = 0
for x in str:
    if x == "a":
        num_a += 1
print(f"a一共出现了{num_a}次")
```

## 06 range语句

> * 在for循环的待处理数据集中，严格说来，是一个序列类型，其内容可以一个个依次取出来，包括：字符串、列表、元组等等
> * range(num)即**获取一个从0开始，到num-1的一个数字序列**
> * range(num1, num2, step),一般来说，都是**左闭右开的，所以取不到num2，step步长默认为1**
```python
for x in range(10):
	print(x)
	
for x in range(5, 10):
	print(x)
	
for x in range(5, 10, 2):
	print(x)
```
```python
# 判断从1到num中一共有多少个偶数
num = 20
count_even = 0
for i in range(1 , num+1):
    if i % 2 == 0:   #偶数的特性就是对于2取余为0，这里是判断符号，不是赋值符号
        count_even += 1
print(f"从1到{num}中，一共有{count_even}个偶数")
```

## 07 变量作用域

> * 在for循环中，临时变量在编程规范上，作用范围（作用域）只限定在for循环内部
> * 但是在实际中，也可以在外部访问到临时变量，但不符合编程规范
```python
for i in range(5):
	print(i)
print(i)  # 这样写不符合规范，因为在外部尝试访问内部临时变量

# 正确写法
i = 0
for i in range(5):
	print(i)

print(i)  # 其实是借助了第一行的变量i，通过for循环
```
## 08 for循环嵌套

> * for循环和while循环也可以相互嵌套
> * 若外层循环为`m`次，内层循环为`n`次，则一共执行`m*n`次

```python

i = 1
for i in range(1,101):

print(f"这是第{i}天，表白成功")
```

## 09 循环中断：break和continue

> * 在循环中可能会出现需要**跳过某次循环或是提前退出循环**的情况，则需要通过continue和break关键字处理
> * **continue中断所在循环的当次执行，跳回循环开始处，直接进入下一次**
> * **break会结束所在的循环，无论剩多少次循环，如果循环有else子句，也会跳过**
> * 也可以用在嵌套循环中，**但只影响所在层循环的继续或终止**
```python
for i in range(1, 100):
# 当代码执行到continue时，不会再执行代码2，而是结束本次循环，进入下一次循环（这里有99次）
	# 这里是代码1
	continue
	# 这是代码2

for i in range(1,100):
"""在这里，代码3不执行，其余代码都会执行。按顺序的话，先代码1，然后进入内层循环，循环执行代码2，然后进入外层循环"""
    #代码1
    for j in range(1,100):
        #代码2
        continue
        #代码3
    #代码4

for i in range(1,100):
"""continue是跳过之后的进入下一次循环，而break是直接退出循环。在执行一次代码1后，执行代码3"""
    #代码1
    break
    #代码2
#代码3

for i in range(1,6):
"""这段代码的执行过程是代码124,124,124……共5个循环"""
    #代码1
    for j in range(1,6):
        #代码2
        break
        #代码3
    #代码4
    
# 例子一
times = 3
for i in range(times):
    name, password = input("请输入用户名："), input("请输入密码：")
    if name == "无极" and password == "666":
        print("登录成功")
        break
    else:
        print(f"登入失败，您还剩{times - i - 1}次机会")
        
# 例子二

while i <= 4:
    i += 1
    if i == 3:
        continue
    print(i)   # 结果为245,3的情况被跳过了

# 例子三

        if j == 2:
```


### 10.3 实现空心金字塔

```python
level_num = int(input("请输入金字塔的层数："))
for i in range(1, level_num + 1):
    # 打印每一层的前导空格，用于形成金字塔的形状
    print("  " * (level_num - i), end=" ")  # 注意这里是两个空格
    
    # 遍历每一层的每个位置，确定是否打印星号
    for j in range(2 * i - 1):
        # 在每一层的开头、结尾以及最中间的位置打印星号
        if j == 0 or j == 2 * i - 2 or i == level_num:
            print("*", end=" ")
        else:
            # 其他位置打印空格，用于形成金字塔的形状
            print(" ", end=" ")
    
    # 每一层打印完成后换行，为打印下一层做准备
    print()
```

### 10.4 成绩核算

```python
"""
    计算3个班（每班5人）的平均分，还有三个班总共的平均分和及格人数
"""
total_sum, pass_num = 0.0, 0 # 定义总分变量及几个人数
for i in range(3):
    # 定义班级总分变量,因为每个班都要单独计数，所以放在第一个循环内
    per_sum = 0
    for j in range(5):   
        score = float(input(f"输入第{j}个学生的成绩："))
        if score > 60.0:
            pass_num += 1
        per_sum += score
    print(f"第{i}个班级的平均成绩是：{per_sum / 5}")
    total_sum += per_sum
print(f"三个班的平均成绩是：{total_sum / 15}，及格人数为{pass_num}")
```



### 10.5 for else用法

> * **当for循环正常完成遍历，在遍历过程中没有被打断（比如没有执行break语句），则会进入else子句**

```python
# 示例1 else子句会执行
for i in range(3):
    print(i)
else:
    print("没有数字供循环了，遍历结束")

# 示例2 else子句不执行
for i in range(3):
    print(i)
    if i == 2:
        break
else:
    print("没有数字供循环了，遍历结束")
```
