### 02.1 文件路径的拼接
> * 可以使用`/`运算符来拼接文件的路径
```python
# 示例：将拼接好的文件路径直接作为参数传入读取方法中
import pandas as pd
from pathlib import Path

# 定义基础路径
base_path = Path('/home/user/documents')

# 使用 / 操作符拼接子目录和文件名
file_path = base_path / 'subfolder' / 'data.csv'

# 直接使用拼接好的路径读取CSV文件
data = pd.read_csv(file_path)
```

```python
# 打开文件
file = open(name,mode,encoding)
# name为文件的路径，路径前加r不容易报错，可以支持绝对路径和相对路径两种
# mode为打开文件的模式（只读、写入、追加等），r为只读，是默认模式；w为写入，原有内容会被删除；a为追加
# encoding为编码格式

f = open(r"D/测试.txt","r",encoding = "UTF-8")

f.read(num)   #num可以指定读取的字节数，不填的话，为读取全部内容

f.readlines()   # 读取文件全部行，封装进一个列表中。前面读过的话，也会从上一次结尾开始读
f.readline()   # 只读取一行

# for循环读取文件行，每次获取一行的内容
for line in f:
    print(f"每一行数据是：{line}")

f.close()

with open(r"D/测试.txt","r",encoding = "UTF-8") as f:
    #这里写文件操作代码
```

> * 文件的写入（**模式为 w** ）

```python
# 打开文件
f = open("D:/wr.txt","w")   # 这个文件是不存在的，相当于这里是新建了一个文件
# 如果打开的是存在的文件，write方法会覆盖掉原来的内容，写入新内容

# 文件写入
f.write("gqergte")
# write并没有把内容写进文件，而是先存在内存之中，称之为缓冲区

# 内容刷新
f.flush()
# flush后内容才会真正写入文件，这样可以避免频繁操作硬盘导致效率下降
f.close()   # close内置了flush功能，所以不写flush也可以
```

> * 文件的追加（**模式为 a** ）

```python
# 打开文件，不存在的文件
f = open("D:/test.txt","a",encoding = "UTF-8")
# write写入
f.write("gerwt")   # 此时文件内容为gerwt
# flush刷新
f.flush()
f.close()

# 打开一个存在的文件
f = open("D:/test.txt","a",encoding = "UTF-8")
# write写入
f.write("发热特发文")   # 若要换行，可在字符最前面加\n
# flush刷新
f.flush()
f.close()   # 此时文件内容为gerwt发热特发文
```

## 03 组织文件 shutil模块

> * shutil模块可以用于文件的复制、移动、重命名和删除等操作
>
> * 在Windows系统中，路径的分隔符通常使用反斜杠`\`
>
> * 在Python中，`pathlib`库和`shutil`模块都支持使用正斜杠`/`或反斜杠`\`来表示路径，但在书写路径时，为了保持与Windows系统的一致性，使用反斜杠更为常见
>
> * 然而，在Python代码中直接写反斜杠需要转义，即使用`\\`。但是，为了简化路径字符串的书写，推荐使用原始字符串（raw string）语法，即在字符串前加`r`，这样就不需要转义反斜杠了
>
### 03.1 复制文件（文件夹）

```PYTHON
# 复制单个文件
import shutil

source = r'C:\Users\username\Documents\source\file.txt'   # 文件地址
destination = r'C:\Users\username\Documents\destination\file.txt'  # 复制到的那个地址
shutil.copy(source, destination)

# 复制整个文件夹到新的文件夹
# 新的文件夹必须不存在
source_dir = r'C:\Users\username\Documents\source_directory'
destination_dir = r'C:\Users\username\Documents\destination_directory'
shutil.copytree(source_dir, destination_dir)

# 复制文件夹到一个已存在的文件夹中
import os
import shutil

source_dir = '/path/to/source/directory'
destination_dir = '/path/to/destination/directory'

# 确保目标目录存在
import os
import shutil

source_dir = '/path/to/source/directory'
destination_dir = '/path/to/destination/directory'

# 确保目标目录存在
os.makedirs(destination_dir, exist_ok=True)

for root, dirs, files in os.walk(source_dir):
    for file in files:
        src_file = os.path.join(root, file)
        dst_file = os.path.join(destination_dir, os.path.relpath(src_file, source_dir))
        shutil.copy2(src_file, dst_file)
```

### 03.2 移动与重命名文件（文件夹）

> * 在使用`shutil.move()`时，如果目标路径指向的是一个存在的文件或文件夹，它将会被覆盖
> * 如果目标路径不存在（例如，目标文件夹不存在），`shutil.move()`会尝试创建必要的中间目录

```python
import shutil

# 假设你有文件夹位于 /path/to/source/folder
source_folder_path = "/path/to/source/folder"

# 移动并重命名文件夹到 /path/to/destination/newfolder
destination_folder_path = "/path/to/destination/newfolder"

shutil.move(source_folder_path, destination_folder_path)
```



## 04 文件操作综合例子

```python
"""
文本内容为:name，date,money,type,remarks
例如:周杰伦,2022-01-01,10000,消费,正式

任务为将文件写出到bill.txt.bak文件作为备份，同时将文件内remarks为测试的数据行删除
"""
# 打开文件
fr = open("D:/bill.txt","r",encoding = "UTF-8")   # 模式r为只读
fw = open("D:/bill.txt.bak","w",encoding = "UTF-8")   # 新文件，模式w为写入
for line in fr:
	line = line.strip()   # 因为有换行符，用strip可去掉
	if line,split(",")[4] == "测试"：
    # 因为内容是以逗号分隔的，所以可以切分后放进一个列表，remark是索引为4的元素
		continue   # 我们要把正式的找出，测试的就跳过，进入下一行判断
	fw.write(line)
	fw.write("\n")   # 因为前面strip了换行符，现在加回去
fr.close()
fw.close()
```
