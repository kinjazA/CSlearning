# 01 函数的多返回值

> * 对于下面的函数，其返回值为1，因为执行完第一个return语句后，便会结束该函数，导致后面的代码不执行

```python
def return_num():
    return 1
	return 2
```

> * 若要返回多个值，可以按下面的写法
> * 按照返回值的顺序，写对应顺序的多个变量进行接收即可，且支持不同类型的数据return

```python
def return_num():
    return 1,2
x,y = return_num()
```

## 02.4 不定长参数

> 函数常见的参数使用方式有4种：
>
> * **1.位置参数**:调用函数时根据函数定义的参数位置来传递参数
>
> * **2.关键字参数**：函数调用时通过"键=值"形式传递参数，不需按照顺序，更加清晰，容易使用
>
>   **若有位置参数，则位置参数必须在关键字参数前，关键字参数不存在先后顺序**，所以也就是第一个参数传递时可以用位置传参
>
> * 位置参数和关键字参数可以混用，不过位置参数要位于关键字参数前
>
> * **3.缺省参数**：也叫默认参数，**所有位置参数必须出现在默认参数前，包括函数定义和调用**,若手动指定，则会将默认的进行覆盖
>
> * **4.不定长参数**：也叫可变参数，用于**不确定调用的时候会传递多少参数**的场景
>
>   类型：1.**位置传递**：传进的所有参数都会被`*args`(规范取名，不强制)变量收集，它会根据传进参数的位置合并为一个**元组**
>
>   ​     2.**关键字传递**：参数是"键=值"形式的情况下，所有键值对会被`**kwargs`(规范取名，不强制)接收，同时组成**字典**

```python
# 位置参数
def user_info(name, age, gender):
    print(f"您的名字是{name},年龄是{age}，性别是{gender}")
user_info("tom",29,"男")   # 传入参数的位置和函数定义的参数要一一对应，位置和顺序不能乱

# 关键字参数
def user_info(name, age, gender):
    print(f"您的名字是{name},年龄是{age}，性别是{gender}")
user_info(age = 20, gender = "男", name = "tom")  
user_info("das",age = 20, gender = "男")    # 这里的第一个参数刚好对应定义的第一个参数，用位置传参

# 缺省参数
def user_info(name, age, gender = "男"):
    print(f"您的名字是{name},年龄是{age}，性别是{gender}")
user_info("fsed",age = 32)

# 不定长参数的位置传递
def user_info(*args):   # *不能少,args是随便取的名字
    print(args)
user_info("tom")   # 只输出tom
user_info("tom",12)   # 输出tom和12

# 不定长参数的关键字传递
def user_info(**kwargs):   # **不能少
    print(kwargs)
user_info(name = "fse",age = 43.id = 1313)

# 标准参数、不定参数在使用时的顺序
def some_func(fargs, *args, **kwargs)
```

# 03 匿名函数

## 03.1 函数作为参数传递

> * 是指函数本身作为参数传入另一个函数内

```python
def func1(compute):   # 函数compute作为参数传入
    result = compute(1, 2)
    print(result)
def compute(x, y):
    return x + y
func1(compute)   # 结果为3
```

## 03.2 lambda匿名函数

> 函数定义
>
> * def关键字，可以定义**带有名称、重复使用**的函数
> * lambda关键字，可以定义匿名函数、**临时使用，仅支持一行代码**，所以适合一些简单的功能

``` python
# 匿名函数的定义语法
lambda 传入参数:函数体(一行代码)   # 传入参数表示匿名函数的形式参数，如x,y表示接收两个形式参数
```

```python
# 通过def关键字定义函数并传入
	"""
	这里的代码逻辑是先把计算逻辑传入func1，再自行定义任意形式的计算逻辑。
	先说明要对数字1,2做一个计算，然后在定义一个计算，传入func1中
	"""
def func1(compute):
    result = compute(1,2)
def compute(x,y):
    return x+y
    print(result)
func1(compute)   # 结果为3

# 通过lambda关键字，取代compute函数
def func1(compute):
    result = compute(1,2)
    print(result)
func1(lambda x,y: x+y)
```
