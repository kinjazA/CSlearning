# 第十二章 闭包

## 01 闭包概念

> * 闭包（Closure）是一个非常强大的概念，它涉及到函数、作用域和变量绑定。闭包**允许内部函数访问并记住其外部作用域中的变量，即使外部函数已经执行完毕**
> * 我理解就是把本来要写成全局变量的放进一个外层函数里，然后在这个外层函数里再写一个函数，使得内层函数可以使用外层函数里的变量。当外层函数执行结束后，内层函数依然可以使用其变量，==**重点应该是外层函数最后是return的内层函数**==
> * 关注点集中在内层函数，对于内层函数来说，闭包的写法下外层函数的变量是一直存在的；而对于外层函数来说，只是一个临时的变量。这样的话，可以避免变量被轻易修改，感觉像是数学上的复合函数

```python
# 示例一
def outer_function(msg):
    # 这是外部函数的一个局部变量
    message = msg

    # 定义一个内部函数
    def inner_function():
        # 内部函数访问了外部函数的局部变量message
        print(message)

    # 返回内部函数，注意这里并没有直接调用inner_function
    return inner_function

# 创建一个闭包实例
closure_example = outer_function("Hello, Closure!")

# 调用闭包实例
closure_example()

# 示例二
def outer(logo):
    def inner(msg):
        print(f"{logo}and{msg}")   # 可以看到这里内层函数使用了外层函数的变量
    return inner
fn1 = outer("蜜雪冰城")             
fn1("我爱你")          # 结果为 蜜雪冰城and我爱你
print(type(fn1))   # 这个fn1其实是函数类型
```

## 02 nonlocal关键字

> * `nonlocal`关键字用于在嵌套函数（即一个函数内部的函数）中修改其外部作用域（但非全局作用域）中的变量。这里所说的外部作用域是指包含当前函数的直接上层函数的作用域，也称为封闭作用域

```python
def outer():
    x = 10  # 这是外层（非全局）变量

    def inner():
        nonlocal x  # 声明x为nonlocal，表示要修改的是外层的x
        x = 20  # 现在修改的是outer函数中的x
        print(x)  # 输出20，因为inner函数改变了outer中的x
    return inner()

fn1 = outer()
```

```python
def account_create(init_amount = 0):
    def atm(num, deposit = True):
        nonlocal init_amount
        if deposit == True:
            init_amount += num
            print(f"存款{num}，账户余额为{init_amount}")
        else:
            init_amount -= num
            print(f"取款{num}，账户余额为{init_amount}")
    return atm
atm1 = account_create()
atm1(600)                  # 结果为：存款600，账户余额为600
atm1(200,deposit=False)    # 结果为：取款200，账户余额为400
```

> 闭包的优点
> * 无需定义全局变量即可实现通过函数，持续的访问、修改某个值
> * 闭包使用的变量作用在函数内，难以被调用修改，安全性更高，也不容易造成错误
> 
> 闭包的缺点
> * 由于内部函数持续引用外部函数的值，所以会导致这部分内存一直被占用
