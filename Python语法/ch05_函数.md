# 第五章 01 函数

## 01 函数的基本语法

> * 函数是组织好的、可重复使用的、实现特定功能的代码段。函数的运用减少了代码的重复性、提高开发效率
> * 传入参数是在函数进行计算时，接受外部提供的数据
> * 在函数调用时，Python的参数传递机制遵循：当你将一个值传递给函数，Python实际上是将该值的引用传递给函数内的局部变量。**对于不可变类型，函数内对参数的修改不会影响到外部的原始值**，因为函数内创建的是一个新的对象。而**对于可变类型，由于对象是共享的，函数内的修改会影响到外部的原始值**
```
def 函数名(传入参数)：
	函数体
	return 返回值
```

```python
def add(x, y):
    result = x + y
    print(f"{x} + {y}的结果是：{result}")
```

> * 函数定义中，x和y称为形式参数，表示函数声明将要使用2个参数；而在实际使用中传入的称为实际参数
> * 形参和实参是一一对应的
> * 函数参数个数不受限制，也可以没有参数，括号内为空

```python
# 一个简单例子
def temp_check(temp):
    if temp <= 37.5:
        print(f"您的体温为{temp}度,体温正常，请进")
    else:
        print(f"您的体温为{temp}度，需要隔离！")
temp_check(40)
```

## 02 函数返回值 return语句

> * 通俗说，返回值就是函数完成一个计算之后，最后给调用者的结果
> * 通过return关键字，就能向调用者返回数据
> * 当希望函数能够与调用它的代码交换信息（通常是计算结果），或者需要在特定条件下提前结束函数执行时，就需要使用`return`语句
> * **同一个函数里，return后的代码不执行**

```python
def 函数名(参数)：
	函数体
	return 返回值或表达式
 
变量 = 函数(实参)    # 函数的调用
```

```python
def sum(x, y):
    result = x + y   # 计算了和，但没有返回

print(sum(12, 45))  # 输出: None

def sum(x, y):
    result = x + y
    return result
print(sum(12, 45))  # 输出：57
```

### 02.1 与break、continue对比

```python
# return的情况
def f1():
    for i in range(1,5):
        if i == 3:
            return  #到达此后的代码不执行
        print(i)
    print("结束了for循环")
f1()                      # 结果为：1 2

# break的情况
def f1():
    for i in range(1,5):
        if i == 3:
            break
        print(i)
    print("结束了for循环")
f1()                    # 结果为：1 2 结束了for循环

# continue的情况
def f1():
    for i in range(1,5):
        if i == 3: 
            continue
        print(i)
    print("结束了for循环")
f1()                    # 结果为：1 2 4 结束了for循环
```

## 03 None类型

> * 没有返回值的函数，实际上就是返回了：`None`这个字面量。函数返回`None`，表示这个函数没有返回什么有意义的内容
>
> * `print()`函数的值就是None类型
>
> * `None`类型的应用场景:
>
>     1.用在函数不用返回值上
>
>     2.用在if判断上，此时`None`等同于`False`
>
>     3.用于声明无初始内容的变量
```python
str1 = print("geargrwea")
if None == str1:
    print("yes")
else:
    print("no")   # 结果为yes，说明print的值就是None类型
```

```python
# 2的用法
def check(age):
    if age > 18:
        print("请进")
    else:
		return None
result = check(16)
if not result:
    """
    因为if后的判断要为True，才会继续执行。None在此时是False，加一个not，就变成了True.
    等价于if result == False:
    """
    print("未成年，不可入内")
    
# 3的用法
name = None
```

## 04 函数的嵌套调用

> * 函数的嵌套调用指的是一个函数里又调用另一个函数
```python
def fun_b():
	print("-----2")
	
def fun_a():
	print("-----1")
	#嵌套fun_b
	fun_b()
	print("-----3")

# 调用函数fun_a
fun_a()  #输出结果为123
```

## 05 函数中变量的作用域
> * 通俗讲，变量的作用域就是指变量在哪里可用，在哪里不可用
> * **局部变量是指定义在函数体内部的变量，只在函数体内部生效**
> * **全局变量是在函数内部和外部都可以使用的变量**
```python
# 出了函数体，局部变量不能使用
def test_a():
	num = 100 # 局部变量
    print(num)
    
test_a()
print(num)  # 报错：num is not defined

# 如果两个函数都使用到一个变量，可以将其单独出函数中，需要时进行调用
num = 200   #全局变量
def test_b():
    print(num)
def test_c():
    print(num)

test_b()
test_c()
print(num)
```

## 06 global关键字

> * 通过`global`关键字，可以将函数体内的局部变量修改为全局变量
```python
num = 200  # 全局变量
def test_a():
    print(f"{num}")   # 输出： 200
def test_b():
    num = 400  # 局部变量
    print(f"{num}")   # 输出： 400，局部变量
print(num)   # 输出： 200，全局变量
    
#将函数体内局部变量转成全局变量
def test_c():
    # global关键字声明num是全局变量
    global num
    num = 400
    print（num）
test_c()   # 输出： 400
print(num)   # 输出： 400，这时不再是200了，因为global全局修改了num
```

## 07 综合例子一

```python
# 一个ATM取款的导入页面
money = 5000000
name = None

name = input("请输入您的姓名：")
# 主菜单函数
def main():
    print("-----------主菜单-----------")
    print("查询余额\t[输入1]")
    print("存款\t\t[输入2]")    #一个制表符对不齐，用两个
    print("取款\t\t[输入3]")
    print("退出\t\t[输入4]")
    return input("请输入您的选择：")
# 查询函数
def query(show_header = True):  # 该参数不传入时默认为True
    if show_header:  
        print(f"-------------查询余额--------------")
    print(f"{name},您好，您的余额为{money}元")
# 存款函数
def save(num):
    global money
    money += num
    print(f"-------------存款--------------")
    print(f"{name},您好，成功存款{num}元")
    query(False)
# 取款函数
def withdraw(num):
        global money
        money -= num
        print(f"-------------取款--------------")
        print(f"{name},您好，成功取款{num}元")
        query(False)
# 设置无限循环
while True:
    keyboard_input = main()
    if keyboard_input == "1":
        query()
        continue  #通过continue继续下一次循环，本次操作已完成
    elif keyboard_input == "2":
        num = int(input("请输入存款金额："))
        save(num)
        continue
    elif keyboard_input == "3":
        num = int(input("请输入取款金额："))
        withdraw(num)
        continue
    else:
        break
```
## 08 综合例子二

```python
# 初始化资金金额
money = 100000
# 初始化通过关卡的数量
pass_num = 0

# 无限循环，用于模拟资金减少直到无法继续通过关卡
while True:
    # 当资金大于50000时，按照5%的比例减少资金
    if money > 50000:
        money -= money * 0.05
        pass_num += 1
    else:
        # 当资金不再满足大于50000的条件时，如果仍大于等于1000，减少资金至1000，并增加通过关卡数
        if money >= 1000:
            money -= 1000
            pass_num += 1
        else:
            # 当资金不再满足大于等于1000的条件时，退出循环
            break

# 输出通过的关卡数量和剩余资金
print(pass_num, money)   #  结果为62 767.4979115529859
```
