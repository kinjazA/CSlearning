# 补充章节一 进制与位运算

## 01 基本介绍

> * 二进制写法：前面加`0b`或`0B`
> * 八进制写法：前面加`0o`或`0O`
> * 十六进制写法：`0x`或`0X` （用0~9和A~F来表示），A是对应10，以此类推

## 02 进制转换

### 02.1 转成十进制

> * `二进制转十进制`：从最低位开始，将每个位上的数提取出来，乘以2的（位数-1）次方，然后求和
> * `八进制转十进制`：从最低位开始，将每个位上的数提取出来，乘以8的（位数-1）次方，然后求和
> * `十六进制转十进制`：从最低位开始，将每个位上的数提取出来，乘以16的（位数-1）次方，然后求和

### 02.2 十进制转其他

> * 可以用`bin()`函数将数字转成二进制
> * `十进制转二进制`：将数字不断除以2，直到商为0，然后将每步得到的余数倒过来（最后一步的写在最前面），就是对应的二进制
> * 可以用`oct()`函数将数字转成八进制
> * `十进制转八进制`：将数字不断除以8，直到商为0，然后将每步得到的余数倒过来（最后一步的写在最前面），就是对应的八进制
> * 可以用`hex()`函数将数字转成十六进制
> * `十进制转十六进制`：将数字不断除以16，直到商为0，然后将每步得到的余数倒过来（最后一步的写在最前面），就是对应的十六进制

### 02.3 二进制转其他

> * `二进制转八进制`：从低位开始，将二进制数每三位一组（最后不够三位就不够），转成对应的八进制数，再拼起来就是对应的八进制的数**（为什么是三位一组，因为三位的二进制刚好能表示八进制的0~7）**
> * `二进制转十六进制`：从低位开始，将二进制数每四位一组（最后不够四位就不够），转成对应的十六进制数，再拼起来就是对应的十六进制的数**（为什么是四位一组，因为四位的二进制刚好能表示十六进制的0~15）**

## 03 二进制在运算中的说明

### 03.1 原码、反码、补码

> * 二进制的最高位是符号位：0表示正数，1表示负数
> * 0的反码、补码都是0
> * 正数的原码、反码、补码都一样（三码合一）
> * 负数的反码 = 原码符号位不变，其他位取反（0变1,1变0）
> * 负数的补码 = 它的反码+1    因而负数的反码 = 负数的补码-1

> * **在计算机执行运算中，是由补码的方式来运算的**
> * 看运算结果时，要转成原码

```python
1 - 3 = 
# 假定使用一个字节来表示数字
 1的补码：0000 0001
-3的原码：1000 0011 反码为：1111 1100 补码为：1111 1101
1 - 3 的补码运算：1111 1110 （这是一个补码）
         反码为：1111 1101
         原码为：1000 0010  这个二进制码对应的十进制数字就是 -2
```

## 04 位运算

> * `~`按位取反，即对**数据的每个二进制位（指补码）取反，1变0,0变1**

```python
# 求 -2 的按位取反
# 还是假定一个字节表示
-2的原码为：1000 0010  反码为：1111 1101 补码为：1111 1110 
 ~-2 ：0000 0001 （这是个补码）    
原码为：0000 0001   对应的十进制数字为1
print(~ -2)    # 结果为1
```

> * `&`按位与，**参与运算的两个值，若相应位都为1，则该位结果为1，否则为0**

```python
# 求2 & 3
2的补码：0000 0010
3的补码：0000 0011
2&3   ：0000 0010
原码为： 0000 0010 对应的十进制数字为2
print(~ -2)   # 结果为2
```

> * `^`按位异或，**当两个对应的二进制位相异时，结果为1，否则为0**

```python
#求2 ^ 3
 2的补码：0000 0010
-3的补码：1111 1101
  2 ^ 3：1111 1111   这是补码
对应原码：1000 0001  对应的十进制数字为-1
print(2^-3)    # 结果位-1
```

> * `|`按位或，**对应的两个二进制位有一个为1时，结果位就为1，否则为0**

```python
# 求2 | 3
2的补码为：0000 0010
3的补码为：0000 0011
   2|3为：0000 0011
 对应原码：0000 0011    对应的十进制数字为3
```

> * `<<`左移，**运算数的各个二进制位全部左移若干位，符号位不变，高位丢弃，低位补0**
> * 左移多少位其实就是乘以几个2，**左移一位就是扩大一倍，乘以2，左移两位，就是乘以2再乘以2**

```python
# 求 5<<1
5的补码为：0000 0101
 左移一位：0000 101    这里把左边高位的0丢弃
 左移一位：0000 1010   这里低位补0
 最终结果：0000 1010
   原码为：0000 1010   对应的十进制数字为10
```

> * `>>`右移，运算数的各个二进制位全部右移若干位，符号位不变，低位溢出，并用符号位补溢出的高位
> * **右移相当于除以2，而且是向下取整**，即整除`//`

```python
# 求 5 << 1
5的补码为：0000 0101
 右移一位：00000 010   对应的十进制数字为2
    
# 求-5 >> 1
-5的补码为：1111 1011
 右移1位为：11111 101   对应的反码为：1111 1100  原码为：1000 0011  对应的十进制数字为-3
```
