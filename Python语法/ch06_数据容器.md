## 01 列表 list

> * 以`[]`作为标识
> * **列表内每一个元素之间用逗号隔开，列表内每一个数据称之为元素**
> * **列表可以存储不同类型的数据，并且支持嵌套，也就是列表里有列表**
> * **允许重复元素存在，可以修改**
```python
# 字面量
[元素1，元素2，……]

# 定义变量
变量名称 = [元素1，元素2，……]

# 定义空列表
变量名称 = []
变量名称 = list()
```
> * **列表的下标索引**：
>
>   元素的顺序是从左到右，**从0开始**，可按照索引来取出特定位置的数据
>   
>   **也可以反向索引，也就是从右往左，右边第一个为-1，然后是-2，以此类推**
```python
my_list = ["xwqeds", 679.7890, True, [3512,353,876,345]]
my_list[0]  # 取出："xwqeds"
my_list[-1][-2]   # 取出：876
```

## 07 序列的切片操作

> * 序列是指内容连续、有序，可使用下表索引的一类数据容器。列表、元组、字符串均可视为序列
> * 切片是指从一个序列中取出一个子序列
> * **语法：序列[起始下标：结束下标：步长]**，**下标还是左闭右开的**
> * 步长为负数表示倒序执行

```python
# 对list进行切片，从索引1开始，4结束，步长为1
list1 = [0,1,2,3,4,5,6]
list1[1:4]   # 步长默认为1，可以省略

# 对tuple进行切片，从头开始，到最后结束，步长为1
tuple1 = (0,1,2,3,4,5,6)
tuple1[:]   # 起始不写表示从头开始，结束不写表示一直到最后

# 对str进行切片，从头开始，到最后结束，步长为2
str1 = "fweagfraewge"
str1[::2]

# 对str进行切片，从头开始，到最后结束，步长为-1
str2 = "01234567"
str2[::-1]   # 这个可以实现翻转字符串的效果

# 对列表进行切片，从3开始，到1结束，步长-1
list1 = [0,1,2,3,4,5,6]
list1[3:1:-1]   # 结果为 3,2,1
```

```python
# 例子练习
```

### 10.1 变量的引用

> * 在Python中，变量名实际上是对象的引用。当你创建一个列表（或其他可变类型的数据结构如字典、集合等）并将它赋值给另一个变量时，实际上是创建了一个新的引用指向同一个对象，而不是创建了一个完全独立的副本。因此，如果你修改了通过另一个引用访问的列表，原始列表也会受到影响，因为它们共享相同的内存空间
>
```python
list1 = [213,5431,654,76534]
list2 = list1
list2[1] = 423
print(list1)
print(list2)
"""
执行list2 = list1时，并没有创建list1的一个全新拷贝给list2，而是让list2指向了list1所指向的同一个列表对象。这意味着list1和list2现在是同一个列表的两个不同名字

通过list2[1] = 423修改列表的第二个元素时，由于list1和list2指向同一块内存地址，这个修改会影响到所有引用该列表的变量

如果想创建list1的一个完全独立的副本，可以使用切片操作list2 = list1[:]或者使用copy()方法list2 = list1.copy()。这样，对list2的修改就不会影响到list1了
"""
```

### 10.2 字典方法实现字符串元素统计

> * `setdefault()`方法允许为字典设置默认值，同时检查该键是否已经存在于字典中。这个方法接受两个参数：一个是键（key），另一个是默认值（default_value）
> * `setdefault()`会尝试查找字典中给定的键。如果找到了该键，就会直接返回该键对应的值，而不做任何其他操作
> * 如果给定的键不存在于字典中，`setdefault()`不仅会将该键添加到字典中，还将其值设置为提供的默认值，然后返回这个默认值

```python
# 给定一个字符串，统计其中全部元素出现的次数
message = "5qerg qgrhvruih, tv2r4oirjv.we -0V438"
count = {}  # 空字典用于计数
for i in message:
    count.setdefault(i,0)
    count[i] += 1
print(count)

import pprint   # 使用该模块可以美观的打印出字典，特别适合字典嵌套字典的输出显示

    count.setdefault(i,0)
    count[i] += 1
pprint.pprint(count)
```
