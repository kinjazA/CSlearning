# 第二章 标识符

## 01 什么是标识符

> 像是变量名字、方法名字、类的名字等都统称为标识符，用来做内容的标识

## 02 标识符命名规则

> * 只允许出现英文、中文、数字、下划线四类元素（**不推荐使用中文、数字不可以开头**）
> * 大小写敏感
> * 不可使用关键字，诸如True、False、for、class等等，但可以包含

## 03 标识符命名规范

> * 变量命名规范：起解释作用、简洁明了、多单词用下划线组合、英文全小写(驼峰形式)          
> * 类名使用大驼峰命名，即多个单词的首字母大写开头                                              

# 第二章 运算符

## 01 算术运算符

> * 常见的加减乘除`/`、取整**（向下）**`//`、取余（取模）`%`、指数`**`，运算顺序与数学里的一致
> * 赋值运算符：`=` ，将右边的结果赋给左边的变量
> * 复合赋值运算符：`+=`，举例：c += a ,等价于c = c + a，还有`-=`,`*=`,`/=`，`%=`，`//=` 等等
> * `a % b = a - a // b * b`

```python
print(-10 // 3)     # 结果为-4，因为是向下取整
print(-10 % 3)      # 结果为2，按照上述公式来进行计算

# 例子：97天转换成几星期零几天
print(f"97天是有{97//7}个星期，零{97%7}天")

# 例子：交换两个数值
# 方法一
x = 132
y = 313
x, y = y, x
print(x, y)

# 方法二
x = y and x
y = x and y
print(x, y)

# 方法三
x, y = 132, 313
x = x + y
y = x - y
x = x - y
print(x, y)
```

## 04 三元运算符

> * 也常称做条件表达式
> * 格式：`value1 = value_if_true if condition else value_if_false`
> * 如例子二，也可以支持嵌套，但过于复杂，一般不建议使用

```python
# 例子一
x = 10  
y = "偶数" if x % 2 == 0 else "奇数"  
print(y)  # 输出: "奇数"

# 例子二
a, b, c = 413, 423, 634
max = a if a > b and a > c else (b if b > a and b > c else c)#先算括号
print(max)
```

# 第二章 字符串

## 01 定义方式

> * 单引号括起
> * 双引号括起
> * 三引号括起，这种写法和注释写法一样，支持换行操作，如果不使用变量接收它，就是注释；若使用变量接收它，就是字符串

## 02 字符串的引号嵌套

> * 单引号定义字符串：可以包含双引号
> * 双引号定义字符串：可以包含单引号
> * 也可以使用转义字符 `\ `将引号解除效用，变成普通字符串，**在要转义的字符前使用**
> * 可以在整个字符串前加`r`使得字符串不会被转义，也就是按字面输出
```python
name = "\"lisa\""

str1 = r"jack\ntom\tgsrdg"   # 此时转义符\n和\t不执行，直接输出
```
## 03 字符串拼接

> * 可以通过加号`+`把两个字符串拼接在一起，加号也可以用于字符串和变量，或变量和变量之间的拼接
> * ==通过加号无法使得字符串和整数、浮点数等进行拼接==
> * **`*`操作符作用在整型值和字符串之间时，会变成字符串复制操作符**
```python
name = "lisa"
print("my name is:" + name + ",i like python.")
tel = 23486896
print("my tel is:" + tel)  # 会报错
str1 = "as" * 3   # 结果为"asasas"
```
## 04 字符串格式化1

> * 可以借助`%`占位符来进行格式化输出
> * 在python中，不可以直接将字符串与数字类型进行拼接，**可以借助str()将数字转成字符再拼接**
> * 可以实现将字符串与数字或其他类型进行拼接
```python
name = "lisa"
message = "my name is : %s" % name
# 字符串里的%表示占位，s表示将变量变成字符串放入占位的地方，即把后面的变量name放入

num1 = 74
num2 = 6000
message = "the first number is %s，and the second number is %s" %(num1, num2)
print(message)
```
> * 其余占位符：%d，将内容转换成整数，放入占位位置；%f，将内容转换成浮点数，放入占位位置

## 05 字符串格式化2（推荐）

> * 通过语法：`f"内容{变量}"`这一格式来快速格式化
```python
str1 = "dxeqbhuis"
num1 = 238
num2 = 239
print(f"the string is {str1},and the first number is {num1},the second number is {num2}")
```
## 06 格式化的精度控制

> * 可以使用辅助符号 m.n 来控制数据的宽度和精度
> * **m是控制宽度（若宽度小于数字本身宽度，则该控制不会生效）；.n控制小数精度，会进行四舍五入**
> * 例如：%5d表示将整数的宽度控制在五位，数字11会变成，空格空格空格11；
> * 例如：%5.2f表示将宽度控制为5，小数精度为2，数字11.345变成，空格空格空格11.35

## 07 对表达式进行格式化

> * 表达式是指一条具有明确执行结果的代码语句
```python
print("1*1的结果是：%d" % (1*1)) # 这里直接就是括号内用表达式，不做额外存一个变量
print(f"1*1的结果是：{1*1}")
print("字符串在python中的类型名称是：%s" % type("字符串"))
```
## 08 驻留机制

> * python仅保存一份相同且不变的字符串，不同的值被存放在字符串的驻留池中，python的驻留机制对相同的字符串只保留一份拷贝，后续创建相同的字符串时，不开辟新空间，而是把该字符串的地址赋值给新的变量
> * 在06章节的**变量引用**中也有提及
> * 驻留机制可以避免频繁创建和销毁，利于节省内存

> `驻留机制存在的几种情况：`
>
> * 字符串是由`英文字母大小写，0~9，_`组成的
> * 字符串长度为0或1时
> * 字符串在编译时进行驻留，而非运行时。**也就是说如果有一个变量在运行之后才知晓其值，比如是拼接的，那么即使它的最终值和另外一个变量一样，但也不会存现驻留机制**
> * [-5, 256]的整数数字也存在驻留机制
> * 但在pycharm中，好像只要值相同，就会存在驻留机制，上面的四条是针对原生python解释器的
```python
str1 = "qwe"
str2 = "qwe"
str3 = "qwe"
print(id(str1))    # id函数返回对象（数据）的内存地址
print(id(str2))
print(id(str3))    # 这三个变量的内存地址是相同的 
```
