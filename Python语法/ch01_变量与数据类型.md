# 第三章 数据类型

> * 数据类型这个概念指的是变量所指的内存数据的类型

## 01 整数类型

> * 存储数据的基本单位是字节（byte），而1个字节等于8 bit（位）
> * bit是计算机中最小的存储单位
> * 在python中，`int类型`的字节是**随着数字的增大而增大的，每次增量是4个字节**
> * 可以通过`sys.getsizeof()`来查询对象（数据）的字节大小

## 02 浮点数类型

> * python中支持十进制（用小数点）和科学计数法两种形式表示浮点数
> * 例如：`5.12e2`表示5.12乘以10的2次方；`3.42e-2`表示3.42乘以10的-2次方
> * 浮点类型的计算存在精度损失，可以使用`Decimal类`进行精确计算

```python
from decimal import Decimal
b = Decimal ("8.1")/ Decimal("3")
print("b = ", b)
```

# 第四章 数据类型转换

## 01 隐式类型转换

> * **python变量的类型不是固定的，会根据变量当前值在运行时来选择对应的数据类型**（java、c等语言则没有这一特性），可以通过type()函数来查看其类型
> * 在运行时，会自动向高精度的转换，即整型与浮点计算时，结果会是float类型

```python
var1 = 10
print(type(var1))    # int类型

var1 = 1.1
print(type(var1))    # float类型

var1 = "egts"
print(type(var1))    # str类型
```

## 02 显式类型转换

> * `int()`可以将括号内的变量转成整型类型
> * `float()`可以将括号内的变量转换成浮点类型
> * `str()`可以将括号内的变量转成字符串类型

```python
i = 10
j = float(i)
i = j + 1
print(i)        # 结果为11.0
print(int(i))   # 结果为11
print(type(i))  # 结果为float型，说明int(i)只是调用了i，并没有改变原来的i
```
