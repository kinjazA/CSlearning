# 第十三章 装饰器

## 01 装饰器概念

> * 允许用户在不修改原有函数定义的情况下，给函数添加额外的功能
> * 装饰器本质上是一个接收函数作为参数的函数，它会返回一个新的函数，这个新函数通常会在执行原始函数之前或之后增加一些额外的操作，但最终调用的仍然是原始函数的核心逻辑
> * 使用装饰器可以让代码更加整洁、模块化，便于复用和维护
## 02 装饰器的闭包写法
```python
# 现在想在sleep函数执行的前后各增加一行打印，装饰器就可以在不修改原代码的基础上实现
def outer(func):     # 函数传入的参数是个函数
    def inner():
        print("我睡觉了")
        func()
        print("我起床了")
    return inner

def sleep():
    import random
    import time
    print("睡眠中……")
    time.sleep(random.randint(1, 5))

fn = outer(sleep)     # 实例化这个闭包，并将sleep函数作为参数传入
fn()
```

## 03 装饰器的快捷写法（语法糖）

```python
# 示例一
def outer(func):     # 函数传入的参数是个函数
    def inner():
        print("我睡觉了")
        func()
        print("我起床了")
    return inner

@outer          # 就是把sleep()函数作为参数传入outer()
def sleep():
    import random
    import time
    print("睡眠中……")
    time.sleep(random.randint(1, 5))

sleep()     # 看起来还是在调用sleep()，但是已经增加了功能

# 示例二
def my_decorator(func):
    def wrapper():
        print("Something is happening before the function is called.")
        func()
        print("Something is happening after the function is called.")
    return wrapper

@my_decorator
def say_hello():
    print("Hello!")

# 调用say_hello时，实际上调用的是装饰器返回的wrapper函数
say_hello()

"""
```
